## 一、论命总纲——月令为核

月令是八字的核心点，天地主气的发源。论命首重月令：

- 清纯代表格局能量级别，"独财值千金，独官掌万民"

## 二、体用论命——入手七步法

### 第一步：先分体用（宫位划分）
- **常规**：日时=我宫（体），年月=他宫（用）
- **特殊**：月日同根同旬→月日=我宫，年时=他宫
- 日干=我本人；日柱=我家；日时=我宫（含配偶子女财物思想）

### 第二步：确定命主追求目标
1. 先看日干有无合（合不是财就是官）
2. 看日支有无可用工具（与年月财官产生感应）
3. 看财官状态（旺衰/元神/刑冲/位置）

### 第三步：寻根找出处
- 天干必找地支根源（何处起根源，流向何处住）
- 我宫透出=代表命主说话/工具/特使
- 年月透入=国家社会的东西送来

### 第四步：找渠道看工具
我宫获取财官的6种方式：
1. **合**（三合/六合/暗合/天干五合）
2. **生**（他宫来生我）
3. **制**（克/刑/冲/穿——先定义后制）
4. **墓**（我宫开库收他宫之物）
5. **送**（他宫之气直透我宫）
6. **伏吟**（干支齐心之力）

### 第五步：判断得失
- **得**="我"用什么工具获取了"他"的什么
- **失**="他"用什么工具获取了"我"的什么
- 三种破坏路线：源头/路线/结果被破坏或优化

### 第六步：参考大运
- 大运修复/优化/破坏原局
- 应期在流年

### 第七步：多维验证
- 明看五行，暗推长生
- 神煞空亡参考应用
- 三点一线，两项定一项，方可铁口直断

# 【第三部分：大运流年推演方法】

## 一、命运岁三者关系

```
流年（皇帝·主动·能量最大）→ 大运（臣·被动）→ 命局（百姓·完全被动）
```

- **动则有生克权，静则无生克权**
- 发动的5种关系：合/冲/刑/害(穿)/值(伏吟)
- 进入优先级：合冲为先，刑害次之，生克又次之
- 大运=月令的延伸，运支决定运干和吉凶方向

## 二、流年介入规则——"有根才有作用力"

- 流年干支谁在命局有根→谁才有作用力
- 干有根干重要/支有根支重要/都有根都重要/都无根都无用
- 盖头截脚：只在干支同时有根/无根时才论；单方有根→以有根方决定成败
- 阴阳：阳主克阴主生/同性克力大异性克力小

## 三、大运批断六大功能

1. 引发原局喜用→吉运
2. 引发原局忌神→凶运
3. 引发伏藏五行→暗藏玄机
4. 修改格局成败→职业/财运/婚姻变动
5. 致命局突然失衡→乐极生悲
6. 大运有力三看：有根否/根受伤否/干支协调否

## 四、应期六大规律

1. **找关键字**：原局某五行受伤+岁运同时再伤
2. **命局某字在岁运出现**：该字信息发生变化
3. **地支遁藏在岁运天干出现**：人元透干即应
4. **冲合刑穿墓**：合者主到/冲者主动/墓者主收/穿者主伤
5. **最旺五行为忌再遇生扶**或犯旺→必灾
6. **流通渠道被截断**→凶灾

# 【第五部分：专题分析流程】

## 一、婚姻专题（6步法）

1. **星宫定位**：配偶星+配偶宫+副宫+感应关系
2. **判断早晚婚**：星宫感应强无阻=早；感应弱有阻=晚
3. **判断婚姻质量**：宫位安静无刑冲穿暗合=好婚
4. **判断多婚/独身**：星宫相战/多现杂现/合走=多婚；比劫占位/星宫相破/财官入墓=独身
5. **看外遇**：财旺/劫旺/食伤旺/金水旺/合多/火土燥
6. **合婚**：年命纳音→年柱正五行→日柱天干→属相→神煞→喜用互补

**关键口诀**：
```
男女婚前看月令，遇合遇冲动感情
流年六合来串宫，也主当年婚姻动
配偶被冲遇着合，配偶被合遇着冲
```

## 二、财运专题（5步法）

1. **定位财星**：以何十神为财（财星/食伤/官杀/禄）
2. **看感应**：财与命主有无必然联系
3. **看渠道工具**：6种获取方式是否成立
4. **判断级别**：巨富/大富/中富/小富（看带库否/旺衰/元神/瑕疵）
5. **确定应期**：大运引动+流年到位

**财富分类**：
- 以财星看→最基础/级别小
- 以食伤看→靠智慧/级别中
- 以官杀看→靠权力/级别最大
- 财库→存财能力

**求财四模式**：单位求财/生意求财/技术求财/体力求财

## 三、事业专题（5步断职业法）

```
Step 1：我宫有官杀？→此柱为职业柱
Step 2：我宫无官杀有官杀库？→库柱为职业柱
Step 3：我宫全无？→看他宫官杀与时柱有无关系
Step 4：以上都无？→看财星财库
Step 5：以上全无？→看成格星
```

**职业柱+左柱秘法**：职业柱定"类型"，左柱定"性质"

# 【第八部分：常见断命原则与注意事项】

## 铁口直断的条件
- **三点一线**：至少从三个不同角度验证同一结论
- **两项定一项**：两种以上方法得出相同结论
- 不足两点→不妄断，标注"有此象但需验证"

## 断命禁忌
- 不可仅凭一个神煞下结论
- 不可忽略宫位只看十神
- 不可死搬格局忽视像法
- 不可单看旺衰不看组合

## 太极点转换
- 看事业→以日干为太极
- 看父亲→以财/印为太极重新飞宫
- 看配偶→以配偶星为太极看六亲环境
- 每换一个测项→太极点必须转换

## 盲派 vs 传统命理的冲突处理原则
1. 当五行推理与神煞铁律一致→铁口直断
2. 当两者矛盾→优先以神煞铁律为准（日刃/阴阳差错/铁板煞/三井水等硬性规定）
3. 格局论与体用论不矛盾时并用，矛盾时以体用得失为主
4. 调候论仅在"生不逢时"（寒极无火/燥极无水）时使用

# 【第九部分：胎身命与长生十二宫】

## 三垣核心规则
- 胎元与年柱不和→骨肉不和/损祖离地
- 命宫与日支或年支犯刑冲→贫贱体弱短寿
- 命宫地支被月支冲克→一举成名（反之大凶）
- **命宫与四柱任何一柱天克地冲/天同地同→该柱限内必有生死大灾**

## 长生十二宫实战核心
- **明五行，暗推长生**（绝招）
- 以日干对照四支→看各宫位六亲/命主在各阶段的状态
- 以六亲天干为太极→对照大运地支→看六亲在大运中的处境
- 沐浴+空亡=桃花无人管
- 妻星到妻宫为胎=未婚先孕
- 日主禄冲夫星禄=嫁娶难

## 马倒禄斜判死术
```
日干起禄（顺行）+ 时干起马（逆行）
查禄马起数：甲己5岁壬申/乙庚4岁辛巳/丙辛3岁丙申/丁壬2岁辛亥/戊癸1岁庚寅
加年法：阴加4/阳加3/卯酉加6
禄马同槽=干支相同
马倒禄斜=干支天克地冲
禄马支到长生地=命归天
```
