### A.1.1 排盘自证清单

排盘必须当场可验证，最小集合如下，缺一作废重排：

| 自证项 | 说明 |
|---|---|
| 干支四柱 | 年/月/日/时，按节气换月 |

### A.1.2 力量对比（粗判，只看月令）

| 判定项 | 标记 | 触发条件 |
|---|---|---|
| 日主得令 | ✓ | 日主五行 = 月令本气 / 月令生日主 |
| 日主失令 | ✗ | 月令克日主 / 月令泄日主 |
| 中和 | ◯ | 月令与日主比劫同气但非本气 |

> 这一步只做"得令 / 失令 / 中和"三档**粗判**，不做"身强身弱百分比"细算（细算交给模块 B）。

### A.1.3 基础作用关系（只列已成立项）

只输出**已成立**的作用，不要列"潜在但未引动"的作用，更不要给效率分。

| 作用 | 必要条件（成立才列） | 输出格式 |
|---|---|---|

## 【A.2 进阶节】（Step 3 按需 · 用户追问"作用细节"才加载）

### A.2.1 穿的判读细则

穿不是单纯"凶"，而是**暗损 / 消耗 / 暗算**类做功。判读三问：

1. **穿到哪根**？穿日支 → 自身 / 穿月令 → 根基 / 穿时支 → 子女晚景
2. **穿用神还是穿忌神**？穿用神 → 坏功 ；穿忌神 → 反成
3. **是否有解**？被合住 / 被冲开 / 距离远 → 力度递减

| 穿的类型 | 性质 | 默认效率（无救解） |
|---|---|---|
| 子未穿 | 阴湿穿 | 中（伤情) |
| 丑午穿 | 火土穿 | 中高（伤心血/官印) |
| 寅巳穿 | 木火穿 | 中（动而生事) |
| 卯辰穿 | 木土穿 | 高（伤夫妻宫常见) |
| 申亥穿 | 金水穿 | 中（伤食伤/口舌) |
| 酉戌穿 | 金土穿 | 高（伤库 / 伤晚运) |

### A.2.2 破的判读细则

破 = 表面成、实际不成。常见两类：

- **破合** → 已合的两支被第三字破开（拆散）
- **破刑** → 已成刑的格局被破打散（救解）

破在原局往往**轻**，破在大运流年才会**重**——这一点要在模块 C 中校验。

### A.2.3 墓库判读

库支：辰（水库 / 戊土库）、戌（火库 / 戊土库）、丑（金库 / 己土库）、未（木库 / 己土库）。

| 入库情形 | 含义 |
|---|---|
| 财入财库 + 库不被冲 | 财不显但实有 |
| 财入财库 + 库被冲开 | 库门被开 → 求财动作发生年 |
| 官入官库 + 库被冲开 | 升迁 / 调动年 |
| 比劫入库 | 兄弟朋友受伤、合作破裂 |
| 印入印库 | 学历 / 母亲事 |
| 食伤入库 | 才华被压、子女事（女命） |

> 墓库的"开 / 闭"由**冲**触发，因此判读必须与模块 C 配合（看哪一年来冲）。

### A.2.4 合的判读细则

#### 天干五合
| 合 | 化神 | 化成条件 |
|---|---|---|
| 甲己合 | 化土 | 月令辰戌丑未或火土旺 |
| 乙庚合 | 化金 | 月令申酉巳或金气旺 |
| 丙辛合 | 化水 | 月令亥子或水气旺 |
| 丁壬合 | 化木 | 月令寅卯亥或木气旺 |
| 戊癸合 | 化火 | 月令巳午寅或火气旺 |

化神**得令** = 真化（合得成）；化神**失令** = 假合 / 空合（合得不成）。

#### 地支六合
寅亥 / 卯戌 / 辰酉 / 巳申 / 午未 / 子丑

判读看是否**同气相求**：
- 同气（如卯戌合化火）→ 实合
- 冰炭相犯（如巳申，火金相战）→ 空合，貌合神离

#### 三合局
申子辰水 / 亥卯未木 / 寅午戌火 / 巳酉丑金。
- 三字俱全 → 局成
- 两字 + 第三字在大运/流年 → 待引动（→ 模块 C）
