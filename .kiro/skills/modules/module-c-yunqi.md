## 【C.2 流年-大运四种交互（Step 3 深推）】

> ⚠ **进入 C.2 之前的强制反向校验（v6.0.3 新增）**：
> 模块 A 给出的大运表第一步起岁，必须与 `module-a-paipan.md § A.1.1.1` 第三连自证一致。
> 若不一致（例如自证显示"5岁1月起运"但大运表写"6岁起"）→ **拒绝进入 C 的应期推演**，回到模块 A 重排。
> 这一条是 v6.0.3 排盘地基补强的硬约束，**不允许跳过**。

### C.2.1 交互类型矩阵

不只看流年好坏，更看流年与**所处大运**的关系：

| 交互类型 | 定义 | 兑现度系数 |
|---|---|---|
| **动** | 流年冲 / 合大运的关键字 → 激活该运能量 | ×1.3 |
| **拦截** | 流年克 / 穿大运用神 → 该运能量被压制 | ×0.6 |
| **兑现** | 流年与大运同气 → 大运承诺照单兑现 | ×1.5 |
| **空转** | 流年与大运无明显作用 → 该运按基础值发挥 | ×1.0 |

### C.2.2 流年判读三步

```
Step 1: 流年干支 vs 原局：是否冲 / 合 / 穿 / 入墓 / 透出？
Step 2: 流年干支 vs 大运：四种交互类型哪一种？
Step 3: 综合 → 触发什么"动作"（婚 / 财 / 病 / 工作 / 子女 / 父母）
```

> 三步都要走完。**只看流年不看大运**是 v5.0 之前最常见的错。

### C.2.2.1 应期层级锚定规则（v6.0.1 新增 · 来自 #CASE-001 REVISE）

> ⚠ 在判定"这一年兑现什么事件"之前，**必须先与模块 B 的格局判定做交叉验证**：

| 模块 B 判定 | 应期锚定规则 |
|---|---|
| 格局标"**轻松兑现**"（如"丙子运轻松兑现"） | 应期优先锚在**大运前 1/3 段** |
| 格局标"**有望但需触发**"（如"乙亥运有望，需某年甲寅"） | 应期锚在**大运中段偏后** |
| 格局标"**极难**"（如"副处极难"） | **不给具体年份**，仅给"大运窗口"概率 |

**铁律**：如果模块 B 已判某目标"轻松"，模块 C **不得**把应期推到该大运的最后 1–2 年——否则构成"跨模块逻辑矛盾"。

**违规案例**：#CASE-001 原判"副科轻松兑现"却锚在丙子运末期 2026 年，实际副科在 2021–2025 已落地。

## 【C.3 关键年份判定（不要随便标"重大事件年"）】

某年成为"重大事件年" **必须同时满足三条**：

1. **兑现度 ≥ 1.3**（即"动"或"兑现"型）
2. **三点验证 ≥ 2 点同向**（干支作用 / 宫位 / 十神 中至少 2 点指向同一事件）
3. **大运能量本身不为负**（即不在"全逆"运）

> 三条同时满足 → 给概率分值 ≥ 70 ；否则只能给 ≤ 60，并写"需进一步触发"。

### C.3.1 应期事件类型清单

应期事件必须从下表中选定一类，不允许"含糊地说今年大吉"：

| 类别 | 触发信号关键 |
|---|---|
| 婚（成 / 离） | 配偶星合日 / 夫妻宫被冲 + 流年同气 |
| 生育 | 食伤入命宫 / 子女宫被合 / **或 已婚态下，子女宫被冲（如丑未冲）+ 食神 / 正印同字共振（来自原局子女宫藏干）**（v6.1.0 升级，详见 §B.9 伏吟铁证识别） |
| 财（得 / 破） | 财星入库被冲 / 群比劫财 / 财坏印 |
| 工作（升 / 降 / 调） | 官星动 / 官入库被冲 / 伤官见官 |
| 健康 | 日主受重克 / 用神受穿破 |
| 学业 | **印透干 + 印得令** / 文昌动（注意：印仅藏不透时不能直接归此类） |
| 父母 | 印星受冲 / 偏正印分别对应 |
| 子女 | 食伤宫动（女命）/ 官杀动（男命） |
| 官非 | 七杀无制 / 伤官见官 + 大耗 |
| 搬迁 / 远行 | 驿马动 + 时柱受冲 |
| **兴趣分流 / 跨界 / 副业**（v6.0.2 新增） | 劫财/比肩大运 + 印不透 + 伤官旺 → 能量出口偏向"非主线"事项（体育、社团、副业、跨界） |

### C.3.2 应期归类力量优先级（v6.0.2 新增 · 来自 #CASE-001 REVISE-002）

⚠ 同一大运可能同时具备多种十神含义（如戊寅大运 = 戊劫财透 + 寅中藏甲丙戊），应期归类必须按下面**力量优先级**判断：

```
透干 > 本气 > 中气 > 余气 > 藏支
```

**判定流程**：

```
Step 1: 看大运天干透出什么十神 → 这是该运的"主能量"
Step 2: 看大运地支本气藏什么 → 这是该运的"次能量"
Step 3: 看原局对应十神是否透干 → 透 = 该应期类型可激活；不透 = 仅潜伏
Step 4: 优先级排序，最高力量者决定应期归类
```

**反面案例（不要重蹈）**：
- 戊寅大运 + 原局印仅藏不透 → 错误归类"印运 = 学业开窍"；正确归类"劫财领先 = 兴趣分流"
- 庚辰大运 + 原局食伤透 → 错误归类"食伤运 = 创业"；正确看"庚伤官 vs 辰中藏戊劫"哪个透根更稳

### C.3.3 三合拱印局应期权重提升（v6.0.4 新增 · 来自 R-002-ADD-03）

⚠ **大运三合拱印局的实际印力远大于"单字透印"** —— 模型必须显式识别拱局，否则会严重低估学业/官印兑现能力。

**触发条件**（满足任一即视为"拱印局激活"）：
1. 大运地支 + 流年地支 + 原局某支 → 构成**三合局**（亥卯未 / 寅午戌 / 巳酉丑 / 申子辰）
2. 三合局所化之神为命主**印星**或**官星**
3. 拱出的"中神"（如亥卯未拱卯）为命局原本不显或藏弱的关键用神

**应期权重系数（v6.0.4）**：

| 印星介入方式 | 权重系数 |
|---|---|
| 单字透印（仅大运天干透印）| ×1.0 |
| 单字透印 + 印得令（大运地支为印） | ×1.3 |
| **三合拱印局激活**（大运 + 流年 + 原局共拱）| **×1.6** |
| **三合拱印 + 印再透流年**（双重激活）| **×2.0**（学业兑现最强信号）|

**典型应用**：
- #CASE-003 丁亥大运（12-22 岁，含 2018 高考）：原局**未支为乙木库**、年支辰为湿土生木，大运**亥**与原局**未**共同与流年隐含"卯"位拱出**亥卯未**三合木局 → **拱出强力卯木印** → 表层"印虚透"被实际兑现为顶级学历

**铁律**：识别拱印局后，模块 B 的学历预测层次**至少提升一级**。

### C.3.4 食神/伤官透干年 + 日主墓库填实 = 才能爆发兑现年（v6.0.4 新增 · 来自 R-002-ADD-02）

⚠ 食神透干流年的爆发力，叠加日主墓库填实时被严重低估。

**触发条件**（同时满足 ≥ 3 项）：
1. **流年天干**透出**食神或伤官**（吐秀 / 才能外显）
2. **流年地支**为**日主之墓库**（甲乙木墓未 / 丙丁火墓戌 / 戊己土墓辰 / 庚辛金墓丑 / 壬癸水墓辰）
3. 大运**透出劫财/比肩**（帮身让食伤吐秀有底气）
4. 流年地支与原局有**冲合关系**（激活 / 填实 / 拱出某神）

**应期类型**：
- 主类：**学业兑现**（高考/考研/考博上岸）
- 次类：**才能爆发**（作品/项目/比赛获奖）
- 兼类：**身份升级**（升职/转岗/获认证）

**典型应用**：
- #CASE-003 2018 戊戌年：戊（食神）透干 + 戌（丙日主火墓）+ 戌冲辰（食神宫被冲激活）+ 大运丁亥丁火帮身 + 亥卯未拱印 → **701 分上浙江大学（C9）**

**铁律**：识别此组合时，该年学业/才能类应期权重 **×1.5**。

## 【C.3.5 大运地支同序共振 = 同频夫妻（合婚专属应期）（v6.1.0 新增 · R-002-COMPAT-02 · 0.6/2.4 黄灯）】

> 来源：`案例库/案例002_夫妻合婚深度分析_2026-05-16.md` 第一卷 §2.3『大运高度同步（罕见）』+ `案例库/案例002_婚育时间线完整应验_2026-05-16.md` 二 §2.1『双方"丑运共振"——命定结婚窗口』。

### C.3.5.1 概念

```
单盘大运：当事人本人未来几十年的地支节奏
合婚大运同序共振：双方各自一生大运地支按"相同顺序"行进
                → 一生节奏共振（结婚 / 育子 / 升迁 / 中年危机 / 退休皆同步）
                → 全人口 < 1% 的命定级同步婚信号
```

### C.3.5.2 触发条件（合婚专属）

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §C.3.5-A：双方大运表前 5 步地支 ≥ 4 步同序 / 同名 / 同冲              │
│                  → 视为"大运地支同序共振"                                   │
│  铁律 §C.3.5-B：识别后必须给出"共振婚育期" / "共振高峰期"具体年份段         │
│  铁律 §C.3.5-C：与模块 D §D.4 联动（同步性是 D.2 维 6 评分依据）            │
└──────────────────────────────────────────────────────────────────────────┘
```

| 同步级别 | 命中数（前 5 步大运地支） | 评级 |
|---|---|---|
| ★★★★★ 命定级共振 | 5/5 全同序 | 案例 002 已应验 |
| ★★★★ 强同步 | 4/5 同序 | 同居 / 领证 / 育子 / 升迁多窗口共振 |
| ★★★ 中同步 | 3/5 同序 | 5-10 年共振段 |
| ★★ 弱同步 | 2/5 同序 | 仅个别窗口同步 |
| ★ 无 | ≤ 1/5 同序 | 无大运同步性，不适用本节 |

### C.3.5.3 子规律：双方同走某地支大运 = 共振婚育期（R-002-COMPAT-06）

> 即使整体大运不全同序，**只要双方某一段大运地支相同**（如双方同走丑运、同走子运），该段就是"共振婚育期"。

**触发条件**：
- 双方某 10 年大运段地支相同（如男方 23-32 走丑、女方 22-31 也走丑）
- + 该地支与双方夫宫 / 子女宫有作用关系（合 / 冲 / 刑 / 半合）

**应期权重**：×1.3（在该共振段内的婚 / 育 / 共同事业概率提升 30%）

**实战兑现**：
- 案例 002 男方 17-26 岁癸丑运 + 女方 20-29 岁辛丑运 → **2018 戊戌年双方丑运共振 → 领证应验**
- 同段后续：2021 辛丑年伏吟双方丑运 → 育长子应验

**规律编号**：**R-002-COMPAT-06**『双方同走某地支大运 = 共振婚育期』 0.6/2.4 🟡

### C.3.5.4 案例锚点（案例 002 五步同序）

```
              男方一生大运地支：寅 → 丑 → 子 → 亥 → 戌 → 酉 → 申
              女方一生大运地支：辰 → 巳 → 午 → 未 → 申 → 酉 → 子 → 亥 → 戌 → 酉
                                                              ↑
                                              从 30 岁起：子 → 亥 → 戌 → 酉

              → 4/5 同序（子→亥→戌→酉）= ★★★★ 强同步级
```

> 双方一生从 28-30 岁起 4 步大运地支同序前进 → 命定级共振。详见 #[[file:.kiro/skills/modules/module-d-hehun.md]] §D.4.2 大运地支同序共振表。

## 【C.3.6 官财同字呼应 = 一字两用（合婚专属规律）（v6.1.0 新增 · R-002-COMPAT-03 · 0.6/2.4）】

> 来源：`案例库/案例002_夫妻合婚深度分析_2026-05-16.md` 第一卷 §2.2『官财同字（互为对方所求）』+ 第二卷论断四。
> **本节定位**：C 模块负责把"官财同字"识别后转化为应期触发系数（×1.5）。详细的合婚静态评分见 #[[file:.kiro/skills/modules/module-d-hehun.md]] §D.5。

### C.3.6.1 概念

```
官财同字 = 男命妻星（财星）的字 = 女命夫星（官杀）的字
       → 在同一个干支字内，男看到妻 / 女看到夫
       → "互为对方所求"

应期含义：当流年透出该共同字时
       → 男方"妻星动" + 女方"夫星动" 同时发生
       → 双方在同一年内对彼此关系产生重大反应
       → 触发力度 ×1.5（合婚专属系数）
```

### C.3.6.2 触发条件

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §C.3.6-A：必须是"字"层面的同字（非"五行"层面的同行）                 │
│                  例：男妻星癸 = 女夫星癸（同字 ★★★★★）                    │
│                      男妻星癸 + 女夫星壬（同行不同字 ★★★，触发系数 ×1.2）│
│  铁律 §C.3.6-B：识别后必须给出"该共同字流年"清单                          │
│  铁律 §C.3.6-C：本节为应期触发系数；静态评分见模块 D §D.5                  │
└──────────────────────────────────────────────────────────────────────────┘
```

| 类型 | 强度 | 应期系数 |
|---|---|---|
| 男妻星字 = 女夫星字 + 同柱位 | ★★★★★ | ×1.5 |
| 男妻星字 = 女夫星字 + 异柱位 | ★★★★ | ×1.4 |
| 男妻星字 = 女夫星五行（不同字） | ★★★ | ×1.2 |

### C.3.6.3 实战兑现（案例 002）

```
男命：戊戌（戌中藏丁戊辛）· 妻星 = 癸（正财，年干透 · "癸酉"）
```

### C.4.5 流年 12 月应期推演法（v6.1.0 新增 · 来源 v2.0 第三部分补丁 A）

#### 六、事件应期到月的『三点一线』月级验证

> **铁律 §C.4.5-B**：要铁口直断某月份应事，必须**同时**满足下面三条：

1. **流年本身支持此事**（大运 × 流年已成气候，先满足 §C.3 三条件）
2. **月干支与事件十神 / 宫位直接对应**（即 5 维表 ① + ②）
3. **月份通用象意 + 神煞填冲 + 起运交运日**相互印证（即 5 维表 ③④⑤）

> 三条同时满足且综合分 ≥ 20 → 标 ★★★★★ 高峰月；
> 否则只能给 ★★★★ 及以下，并写"需进一步触发"。
