> **来源**：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 1 课 §3（"太岁如君，大运如臣，命局好比百姓"）。
> 本节为 §C.1 速览提供**能量层级底层逻辑**，与 §C.2.4 发动五法互补。

```
铁律 §C.1.3-A：三级主被动能量 ——
  流年 = 主动能量最大（如君）→ 发号施令权
  大运 = 次之（如臣）→ 执行/放大/削弱权
  命局 = 被动（如民）→ 被作用方（基因底色）

铁律 §C.1.3-B：被引动后命局亦有生克权 ——
  原局被岁运通过合/冲/刑/害/值发动后，
  命局字照样能反向作用岁运（弱被动变主动）。
  → 不可机械理解"命局无权"，须结合 §C.2.4 发动五法判定。
```

#### 实战要点

| 维度 | 流年（君） | 大运（臣） | 命局（民） |
|---|---|---|---|
| **能量级** | 最高（一年之主） | 中（十年之运） | 最低（静态基因） |
| **主动性** | 主动出击 | 承上启下 | 被动受作用 |
| **作用方式** | 可直接作用命局/可通过大运间接作用 | 放大或削弱流年效应 | 被引动后回击 |
| **联动** | §C.3.10 有根论 | §C.2.4 发动五法 | §C.2.1 交互矩阵 |

## 【C.2 流年-大运四种交互（Step 3 深推）】

> ⚠ **进入 C.2 之前的强制反向校验（v6.0.3 新增）**：
> 模块 A 给出的大运表第一步起岁，必须与 `module-a-paipan.md § A.1.1.1` 第三连自证一致。
> 若不一致（例如自证显示"5岁1月起运"但大运表写"6岁起"）→ **拒绝进入 C 的应期推演**，回到模块 A 重排。
> 这一条是 v6.0.3 排盘地基补强的硬约束，**不允许跳过**。

### C.2.1 交互类型矩阵

不只看流年好坏，更看流年与**所处大运**的关系：

| 交互类型 | 定义 | 兑现度系数 |
|---|---|---|
| **动** | 流年冲 / 合大运的关键字 → 激活该运能量 | ×1.3 |
| **拦截** | 流年克 / 穿大运用神 → 该运能量被压制 | ×0.6 |
| **兑现** | 流年与大运同气 → 大运承诺照单兑现 | ×1.5 |
| **空转** | 流年与大运无明显作用 → 该运按基础值发挥 | ×1.0 |

### C.2.2 流年判读三步

```
Step 1: 流年干支 vs 原局：是否冲 / 合 / 穿 / 入墓 / 透出？
Step 2: 流年干支 vs 大运：四种交互类型哪一种？
Step 3: 综合 → 触发什么"动作"（婚 / 财 / 病 / 工作 / 子女 / 父母）
```

> 三步都要走完。**只看流年不看大运**是 v5.0 之前最常见的错。

### C.2.2.1 应期层级锚定规则（v6.0.1 新增 · 来自 #CASE-001 REVISE）

> ⚠ 在判定"这一年兑现什么事件"之前，**必须先与模块 B 的格局判定做交叉验证**：

| 模块 B 判定 | 应期锚定规则 |
|---|---|
| 格局标"**轻松兑现**"（如"丙子运轻松兑现"） | 应期优先锚在**大运前 1/3 段** |
| 格局标"**有望但需触发**"（如"乙亥运有望，需某年甲寅"） | 应期锚在**大运中段偏后** |
| 格局标"**极难**"（如"副处极难"） | **不给具体年份**，仅给"大运窗口"概率 |

**铁律**：如果模块 B 已判某目标"轻松"，模块 C **不得**把应期推到该大运的最后 1–2 年——否则构成"跨模块逻辑矛盾"。

**违规案例**：#CASE-001 原判"副科轻松兑现"却锚在丙子运末期 2026 年，实际副科在 2021–2025 已落地。

### C.2.4 岁运发动三铁律（v6.1.5 新增 · GP-DY-01 / 02 / 03 合并写入 · 来源高派《大运流年篇》第 1 课 §4~6）

> ⚠ **触发**：高派《大运流年篇》提取报告 GP-DY-01 ⭐⭐ + GP-DY-02 ⭐⭐ + GP-DY-03 ⭐⭐（三条一体 · 候选库 0.6 × 3 → v6.1.5 阶段 5 晋级）。
> 本节是 §C.2 流年-大运交互的**前置铁律层** —— 不满足发动条件则 §C.2.1 四种交互类型**不启动**。
> 来源：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 1 课 §4/§5/§6。

#### 一、发动五法铁律（GP-DY-01 · §C.2.4-A）

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §C.2.4-A：岁运介入命局仅五种方式 —— 合、冲、刑、害、值               │
│                 无此五种关系 → 不发动 → 无生克权 → 该年对该柱无重大作用        │
│                 有此五种关系 → 发动 → 才有资格论生克吉凶                     │
└──────────────────────────────────────────────────────────────────────────┘
```

| 发动方式 | 力度排序 | 典型效应 |
|---|---|---|
| **合**（六合/三合/天干五合） | ①（最强） | 锁定 / 牵引 / 改变性质 |
| **冲**（六冲） | ②（次强） | 激活 / 破坏 / 动荡 |
| **刑**（三刑/自刑） | ③ | 暗伤 / 纠缠 / 法律 |
| **害（穿）** | ④ | 暗损 / 背叛 / 拖累 |
| **值**（伏吟/同字到位） | ⑤ | 共振 / 填实 / 能量倍增 |

> **与 §C.2.1 的关系**：§C.2.1 四种交互（动/拦截/兑现/空转）是**结果分类**；本节五法是**前置条件** —— 先判五法发动与否，再定四种交互类型。

#### 二、岁运进入命局顺序铁律（GP-DY-02 · §C.2.4-B）

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §C.2.4-B：流年 → 大运 → 命局（三级传导）                             │
│                 流年先作用大运，再通过大运进入命局                            │
│                 流年亦可越过大运直接作用命局（条件：五法直达原局某柱）          │
│                 先后顺序 = 合冲为先 → 刑害次之 → 生克又次之                  │
└──────────────────────────────────────────────────────────────────────────┘
```

**判读步骤**：
```
Step 1：流年干支 vs 大运干支 → 有无五法发动？
Step 2：流年干支 vs 原局四柱 → 有无五法直达？
Step 3：大运干支 vs 原局四柱 → 有无五法作用？
Step 4：按"合冲先 → 刑害次 → 生克末"排优先级
Step 5：优先级最高的发动通道 = 该年主事件通道
```

#### 三、大运地支为重铁律（GP-DY-03 · §C.2.4-C）

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §C.2.4-C：大运以地支定吉凶大方向 —— 运支力量决定运干力量               │
│                 运干被运支截脚（甲申 = 甲被截）→ 干弱无用                    │
│                 运干得运支生（甲寅 = 甲得禄）→ 干强有力                      │
│                 判大运好坏先看地支，再论天干                                │
└──────────────────────────────────────────────────────────────────────────┘
```

| 大运组合 | 运干力量 | 判定 |
|---|---|---|
| 干得地支生/禄/旺 | 强 | 运干能量可兑现 |
| 干被地支截脚/克/死绝 | 弱 | 运干有名无实 |
| 干与地支无明显关系 | 中 | 看原局是否有根来接 |

#### 四、与 §C.2.2 流年判读三步的整合

```
新版流年判读流程（v6.1.5 升级）：

        → 是 = 进入 Step 1~3 深推

Step 1：流年干支 vs 原局（哪些柱被发动？合/冲/刑/害/值？）
Step 2：流年干支 vs 大运（四种交互类型 + 地支为重判定）

```

### C.2.5 大运六大功能总纲（v6.1.6 新增 · GP-DY-05 ⭐⭐ · 来源高派《大运流年篇》第 2 课）

> **来源**：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 2 课（一）~（六）。
> 本节是 §C.2.4 发动三铁律的**功能分类层** —— 发动后"这步运在干什么"由本节六大功能定。

| # | 功能 | 含义 | 典型场景 |
|---|---|---|---|
| **① 引发喜用** | 大运干支引出原局喜用神 | 好运开始·能量释放 |
| **② 引发忌神** | 大运引出原局忌神（无根行根运/不透行透运/受制行解救/强旺行忌运/用神被伤） | 坏运开始·灾厄窗口 |
| **③ 引发伏藏** | 原局没有的干支在大运出现 → 暗藏玄机显现 | 突变型事件（贵人/小人突现） |
| **④ 修改格局** | 大运使原局格局成立/被破（成格变格之权） | 人生转折·级别跃迁或坠落 |
| **⑤ 突然失衡** | 身弱突旺 / 身旺突弱 = 乐极生悲或灾祸加重 | 中年危机 / 暴起暴落 |
| **⑥ 有力三看** | 大运是否有力=有根/根未受伤/非死绝 | 判定该运能否兑现承诺 |

**使用方式**：
```
Step 0：§C.2.4 五法发动检查（前置）
Step 1：本节六大功能分类 → 该步运属于哪 1-2 种功能？
Step 2：§C.2.1 四种交互类型判定
Step 3：具体流年深推
```

**铁律 §C.2.5-A**：引发忌神 5 条中任一条命中 → 该运段 **整体标"逆"**（§C.1.1），即使运干表面为喜用也不改标签。
**铁律 §C.2.5-B**：修改格局 = 大运最强功能 → 若该运使命局从"无格"变"有格"（或反向），格局判定（模块 B）必须以该运为基准重做。

### C.2.6 盖头截脚实质判定（v6.1.6 新增 · GP-DY-11 ⭐⭐ · 来源高派《大运流年篇》第 3 课 §一）

> **来源**：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 3 课 §一（"吉凶不验多因此"）。
> 本节解决回归验证中 CASE-001 丙子运 / CASE-003 丁亥运"运干被截脚但实际该运为吉"的判读路径。

#### 一、定义

| 术语 | 含义 | 示例 |
|---|---|---|
| **盖头** | 运干克运支（上克下） | 甲申（甲木被申金截）= 错！应为庚寅（庚克寅）= 盖头 |
| **截脚** | 运支克运干（下克上） | 丙子（子水克丙火）/ 甲申（申金克甲木） |
| **干支顺配** | 运干得运支生/禄/旺 | 甲寅（甲得禄）/ 壬亥（壬得禄） |

#### 二、核心铁律

```
┌──────────────────────────────────────────────────────────────────────────┐
│  §C.2.6-A：盖头截脚判定仅在"都有根"或"都无根"时才有效                       │
│            干有根支无根 → 以干为主，不论盖头截脚                            │
│            支有根干无根 → 以支为主，不论盖头截脚                            │
│  §C.2.6-B：运干被截脚但原局有根来接 → 运干中等可用（非"无用"）             │
│  §C.2.6-C：喜用之运最好干支顺配（如喜木最好甲寅/乙卯，其次壬寅）           │
│            退而求其次=截脚但原局有根 > 盖头无根                            │
└──────────────────────────────────────────────────────────────────────────┘
```

#### 三、与 §C.2.4-C / §C.3.10 的整合

```
判大运好坏完整流程（v6.1.6 升级）：

Step 1：§C.2.4-C 大运地支为重 → 先看运支定大方向
Step 2：§C.2.6 盖头截脚 → 看干支关系：
        - 顺配 → 运干强（运干承诺可兑现）
        - 截脚但原局有根 → 运干中等（可用但受限）
        - 截脚且原局无根 → 运干弱（有名无实）
        - 盖头 → 运支受压（地支能量被干压制，大方向打折扣）
Step 3：§C.2.5 六大功能分类 → 定该运主做什么
```

#### 四、回归验证案例

| 案例 | 大运 | 干支关系 | 原局有根？ | 判定 | 实际 |
|---|---|---|---|---|---|
| CASE-001 | 丙子 | 截脚（子克丙） | 丙在原局有午（帝旺）= 有根 | 运干中等可用 | ✓ 丙子运提拔 |

### C.2.7 引发忌神 5 条铁律（v6.1.7 新增 · GP-DY-06 ⭐⭐ · 来源高派《大运流年篇》第 2 课（二））

> **来源**：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 2 课（二）（"大运引发原局忌神之权"）。
> **状态**：v6.1.7-batch5 候选规则 → 案例003 反向验证条目⑤逻辑通过 → 升级 0.8/2.4 🟡。
> **同步源**：`盲派命理AI分析师提示词_v2.1.md` §C.2.7。

#### 一、5 条铁律

大运引发原局忌神为凶，具体 5 种情形：

| # | 情形 | 判断要点 |
|---|---|---|
| ① | 忌神透干无根，行忌神旺根之运 | 忌神原本虚浮无力，大运给根 = 忌神坐实，凶灾显现 |
| ② | 忌神强旺不透，行透干之运 | 忌神藏于地支蓄势，大运透出 = 暗忌明发，突然应凶 |
| ③ | 忌神已受制服，行解救忌神之运 | 原局有制（合/克/墓），大运冲开制约 = 忌神脱困反扑 |
| ④ | 忌神强旺无制，行忌神大运 | 火上浇油，忌神力量叠加，凶灾最重 |
| ⑤ | 用神/关键字在大运受伤 | 不是忌神变强，而是喜用被破 = 间接引发忌神猖獗 |

## 【C.3 关键年份判定（不要随便标"重大事件年"）】

某年成为"重大事件年" **必须同时满足三条**：

1. **兑现度 ≥ 1.3**（即"动"或"兑现"型）
2. **三点验证 ≥ 2 点同向**（干支作用 / 宫位 / 十神 中至少 2 点指向同一事件）
3. **大运能量本身不为负**（即不在"全逆"运）

> 三条同时满足 → 给概率分值 ≥ 70 ；否则只能给 ≤ 60，并写"需进一步触发"。

### C.3.2 应期归类力量优先级（v6.0.2 新增 · 来自 #CASE-001 REVISE-002）

⚠ 同一大运可能同时具备多种十神含义（如戊寅大运 = 戊劫财透 + 寅中藏甲丙戊），应期归类必须按下面**力量优先级**判断：

```
透干 > 本气 > 中气 > 余气 > 藏支
```

**判定流程**：

```
Step 1: 看大运天干透出什么十神 → 这是该运的"主能量"
Step 2: 看大运地支本气藏什么 → 这是该运的"次能量"
Step 3: 看原局对应十神是否透干 → 透 = 该应期类型可激活；不透 = 仅潜伏
Step 4: 优先级排序，最高力量者决定应期归类
```

**反面案例（不要重蹈）**：
- 戊寅大运 + 原局印仅藏不透 → 错误归类"印运 = 学业开窍"；正确归类"劫财领先 = 兴趣分流"
- 庚辰大运 + 原局食伤透 → 错误归类"食伤运 = 创业"；正确看"庚伤官 vs 辰中藏戊劫"哪个透根更稳

### C.3.3 三合拱印局应期权重提升（v6.0.4 新增 · 来自 R-002-ADD-03）

⚠ **大运三合拱印局的实际印力远大于"单字透印"** —— 模型必须显式识别拱局，否则会严重低估学业/官印兑现能力。

**触发条件**（满足任一即视为"拱印局激活"）：
1. 大运地支 + 流年地支 + 原局某支 → 构成**三合局**（亥卯未 / 寅午戌 / 巳酉丑 / 申子辰）
2. 三合局所化之神为命主**印星**或**官星**
3. 拱出的"中神"（如亥卯未拱卯）为命局原本不显或藏弱的关键用神

**应期权重系数（v6.0.4）**：

| 印星介入方式 | 权重系数 |
|---|---|
| 单字透印（仅大运天干透印）| ×1.0 |
| 单字透印 + 印得令（大运地支为印） | ×1.3 |
| **三合拱印局激活**（大运 + 流年 + 原局共拱）| **×1.6** |
| **三合拱印 + 印再透流年**（双重激活）| **×2.0**（学业兑现最强信号）|

**典型应用**：
- #CASE-003 丁亥大运（12-22 岁，含 2018 高考）：原局**未支为乙木库**、年支辰为湿土生木，大运**亥**与原局**未**共同与流年隐含"卯"位拱出**亥卯未**三合木局 → **拱出强力卯木印** → 表层"印虚透"被实际兑现为顶级学历

**铁律**：识别拱印局后，模块 B 的学历预测层次**至少提升一级**。

> 来源：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 4 课 §一(二)（"流年干支谁在命局有根谁才有作用力…绝招中的绝招"）。

### C.3.10.1 核心铁律

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §C.3.10-A（绝招中的绝招）：                                          │
│    流年天干在命局有根 → 天干重要（天干主事）                                  │
│    流年地支在命局有根 → 地支重要（地支主事）                                  │

```

### C.3.10.2 有根判定细则

| 流年干 | 有根 = 命局中有以下地支 | 无根 = 命局中无此地支 |
|---|---|---|
| 甲 | **寅**（甲木本气禄地） | 卯非根（卯是乙的根） |
| 乙 | **卯**（乙木本气） | 寅非根（寅是甲的根） |
| 丙、戊 | **巳**（丙禄/戊长生） | 午为丁己之根 |
| 丁、己 | **午**（丁禄/己帝旺） | 巳为丙戊之根 |
| 庚 | **申**（庚禄） | 酉为辛之根 |
| 辛 | **酉**（辛禄） | 申为庚之根 |
| 壬 | **亥**（壬禄） | 子为癸之根 |
| 癸 | **子**（癸禄） | 亥为壬之根 |

> **铁律 §C.3.10-B**：三合局的有根以阳干看 —— 如命局有亥卯未三合木局，视为甲木有根（阳干）。
> **铁律 §C.3.10-C**：盖头截脚仅在"都有根"或"都无根"时才看 —— 干有根支无根时以干为主，不必论盖头截脚。

### C.3.10.3 流年无根三种变化

| # | 变化 | 影响 |
|---|---|---|
| ① | 无根但影响环境（财运增减） | 忌神无根=环境差收入少；用神无根=环境好收入多（弱化版） |

| ③ | 通过大运间接作用 | 大运是月令延伸，流年作用大运=间接影响命局（不论有无根都有用） |

```
│                   乙以卯为根 + 辰未亦可 / 辛以酉为根 + 丑戌亦可            │
│  铁律 §C.3.12-C：三合局以阳干看                                           │
│                   命局有亥卯未任二 → 三合木局 → 流年甲乙同做有根看          │
│                   大运补足三合第三字 → 流年阳干也有根                       │
└──────────────────────────────────────────────────────────────────────────┘
```

### C.3.12.3 三合局补根详例（木为例）

```
情形 1：命局有卯+未 → 乙木有根（卯=禄）
        大运逢亥 → 亥卯未三合 → 甲木也有根（三合以阳干看）
        → 流年甲乙均有根

情形 2：命局仅有未（无卯无亥）→ 乙有根（未=余气）/ 甲无根
        大运逢卯 → 卯未半合 → 乙根更强，甲仍无根（半合不足以成局）
        大运逢亥 → 亥未拱卯 → 视具体情况（原则上拱局弱于实合）

情形 3：命局有亥+卯 → 亥卯半合木 → 甲有根（亥为甲长生+卯为半合）
        大运逢未 → 亥卯未三合全 → 甲乙均强根
```

### C.3.12.4 与 §C.3.10 的关系

§C.3.10.2 给出简表（仅禄地），本节为**完整细则**。当 §C.3.10 判"无根"时，须来本节二次检查：

- 大运是否已补足三合第三字？

## 【C.3.13 流年无根三种变化（v7.0 新增 · GP-DY-18 ⭐⭐ · 来源高派《大运流年篇》第 4 课 §一(二)）】

> **来源**：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 4 课 §一(二)（"第一种…第二种…第三种"）。
> 本节是 §C.3.10.3（已有简表）的**完整展开** —— 无根不等于"无用"，三种变化路径决定实际影响。

### C.3.13.1 三种变化详解

| # | 变化类型 | 机制 | 实战意义 |
|---|---|---|---|
| **①** | **无根但影响环境** | 流年无根 → 不能直接作用命局 → 但改变外部环境 | 忌神无根=环境差收入少但不致灾；用神无根=环境好收入多但不铁断发 |
| **②** | **地支冲合命局（间接介入）** | 天干无根但地支与命局某柱五法发动 | 虽天干无根，地支已介入 → 仍可生吉凶（走 §C.2.4 五法通道） |
| **③** | **通过大运间接作用** | 大运=月令延伸 → 流年作用大运=间接影响命局 | 无论流年有无根，只要与大运有合冲刑害值关系就有用 |

### C.3.13.2 核心铁律

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §C.3.13-A：流年无根 ≠ 流年无用                                      │
│    三种变化路径任一成立 → 该年仍有实际影响                                  │
│  铁律 §C.3.13-B：变化①（环境型）→ 不下铁断，只给"环境好/差"定性           │
│  铁律 §C.3.13-C：变化②（地支间接介入）→ 联动 §C.3.10-D 例外条款           │
│    地支五法直达命局某柱 = 触发器型（可出大事件）                            │
│  铁律 §C.3.13-D：变化③（通过大运）→ 大运是月令延伸                        │
│    流年与大运有五法关系 = 已间接介入命局（不论天干有无根）                   │
└──────────────────────────────────────────────────────────────────────────┘
```

### C.3.13.3 判定流程

```
流年天干无根时的判定路径：

Step 1：检查变化② → 流年地支是否与命局某柱五法发动？
        → 是 → 地支主事（联动 §C.3.10-D 例外路径）→ 可出大事件
        → 否 → Step 2

Step 2：检查变化③ → 流年与大运是否有五法关系？
        → 是 → 通过大运间接作用 → 中等事件（大运能量决定大小）
        → 否 → Step 3

Step 3：走变化① → 仅环境影响
        → 忌神无根=环境差·收入减少（不致灾）
        → 用神无根=环境好·收入增加（不铁断发）
        → 结论：该年影响力弱，不下铁断
```

## 【C.3.14 关键字被克组合断流年（v7.0 新增 · GP-DY-20 ⭐⭐ · 来源高派《大运流年篇》第 5 课 §一.1）】

### C.3.14.1 核心逻辑

```
传统思维：关键字寅 → 被克 → 流年申（六冲）
高派纠正：关键字寅 → 先看是否有合会组合可克 → 合会克优先于六冲克
```

| ① | 有丑 | 酉年 | 巳酉丑三合金局克寅 | ★★★★★ |
| ② | 有巳 | 酉年 | 巳酉丑三合中神酉为重 | ★★★★★ |


## 【C.3.15 保护神阻断克路铁律（v7.0 新增 · GP-DY-21 ⭐⭐ · 来源高派《大运流年篇》第 5 课 §一.2）】

> **来源**：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 5 课 §一.2（"某字与克神之间间隔一个保护神不再适合本思维"）。
> 本节是 §C.3.14 的**例外条款** —— 当关键字与克神之间有保护神时，§C.3.14 的合会克逻辑失效。

### C.3.15.1 概念

```
保护神 = 关键字与克神之间间隔的"我生之神"（泄克神之力）
         → 克神被保护神泄/化 → 无法直接克到关键字
         → §C.3.14 的三合/三会/六冲思维全部失效
```

### C.3.15.2 典型结构

```
例：年丑 · 月午 · 日寅

分析：
- 关键字 = 寅（日支）
- 克神 = 丑（年支 → 丑含辛金克寅木）
- 保护神 = 午（月支 → 午火泄木生土，但更关键是午为寅的"我生之神"产物）

结论：
- 丑与寅之间有午（火）= 保护神
- 巳酉丑三合金局克寅 → 此路被保护神截断 → 不适用
- 永远不会是巳酉丑年或申酉戌年伤此寅字
- 改判：伤寅的方式 = 木年克保护神午（土）→ 保护神失效 → 然后金才能克寅
```

### C.3.15.3 核心铁律

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §C.3.15-A：关键字与克神间有保护神 → §C.3.14 合会克思维完全失效        │
│  铁律 §C.3.15-B：保护神失效年 = 真正应期                                   │
│    保护神被克/被冲/被合去的流年 → 保护神失效 → 克路贯通 → 关键字受灾        │
│  铁律 §C.3.15-C：天干地支同理                                             │
│    天干间有通关之神 → 同样阻断克路 → 判法一致                              │
└──────────────────────────────────────────────────────────────────────────┘
```

### C.3.15.4 判定流程

```
Step 1：§C.3.14 判定关键字被克路径
Step 2：检查 → 关键字与克神之间是否有保护神（间隔字/中间柱位）？
        → 无保护神 → 按 §C.3.14 正常判
        → 有保护神 → 进入本节
Step 3：保护神何时失效？
        - 流年克保护神 → 保护神受伤 → 克路贯通
        - 流年冲/合走保护神 → 保护神离位 → 克路贯通
Step 4：保护神失效年 = 关键字受灾的真正应期
```

## 【C.3.16 远近组合断流年铁律（v7.0 新增 · GP-DY-22 ⭐ · 来源高派《大运流年篇》第 5 课 §二(一)）】

```
远近效应：
```

```
┌──────────────────────────────────────────────────────────────────────────┐

```

## 【C.3.17 人元引出三法（v7.0 新增 · GP-DY-23 ⭐⭐ · 来源高派《大运流年篇》第 6 课（一））】

### C.3.18.2 天干克人元详解（枪打出头鸟）

```

注意：大运与原局同字 → 无真正加力 → 不应灾
  （如丁丑柱行丁运 → 丁与丁同字=不加力→不克辛）
```

## 【C.3.19 交运脱运铁律（v7.0 新增 · GP-DY-14 ⭐⭐ · 来源高派《大运流年篇》第 3 课 §四）】

> ⚠ **高风险提示**：本节含"换好是人换不好是鬼"等极端表述，涉及寿限判断的部分（出生入死）需 ≥ 3 反例校验后方可铁断。日常使用仅取"交运年前后动荡"和"限运对应"两个安全层面。

### C.3.19.1 交运脱运概念

```
交运 = 走完一步大运，进入下一步大运的交接
脱运 = 旧运能量消退，新运能量尚未完全接管

古语："交运脱运两头奔命"
含义：交接期人如脱胎换骨 → 前后 1-2 年最不好过
表现：心神不定 / 性情大变 / 事业感情动荡 / 严重者耗财生灾
```

### C.3.19.2 四大关键点

| # | 关键点 | 内容 | 实战意义 |
|---|---|---|---|
| **①** | 交接之年前后两步大运兼看 | 旧运未尽+新运初至 → 两运能量交织 | 不可只看新运论吉凶，须兼顾旧运余力 |
| **②** | 行运换甲最关键 | "换得好是人，换不好是鬼" → 换甲时五行骤变 | 换甲=天干从癸→甲（水→木）= 五行属性完全改变 |
| **③** | 换甲忌岁运伏吟反吟刑冲 | 交运年如逢岁运并临/伏吟/反吟/刑冲 → 凶灾加重 | 交运+伏吟 = 双重危险 |
| **④** | 大运与限运对应 | 年柱=1-2步 / 月柱=3-4步 / 日柱=5-6步 / 时柱=7-8步 | 大运与对应限运柱发生刑冲穿绝 → 必有大灾 |

### C.4.7 冲合应期互补铁律（v7.0 新增 · GP-DY-26 ⭐⭐ · 来源高派《大运流年篇》第 7 课（四））

> **来源**：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 7 课（四）（"原局有合以冲为应…填实应吉应凶"）。
> 本节是 §C.4.6 应期六大规律第④条"冲合应期互补"的**完整展开** —— 高派定应期的第一法。

#### 一、核心铁律

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §C.4.7-A：原局有合 → 以冲为应期                                     │
│    合住=锁定不动 → 需冲来打开 → 冲年=事件爆发年                           │
│  铁律 §C.4.7-B：原局有冲 → 以合为应期                                     │
│    冲动=不稳定 → 需合来收拢 → 合年=事件落地年                             │
│  铁律 §C.4.7-C：流年合大运 = 合动                                         │
│    合=牵引/锁定 → 大运被流年合住 → 该运能量被引动（吉凶看合方喜忌）        │
│  铁律 §C.4.7-D：流年冲大运 → 分当运/不当运                                │
│    当运（旺相有力）被冲 = 冲去（凶·正在发挥的能量被破坏）                  │
│    不当运（休囚无力）被冲 = 冲起（吉·沉睡的能量被激活）                    │
│  铁律 §C.4.7-E：六合/三合/三会/三刑 → 一字空缺 → 填实应吉应凶             │
│    如亥卯未三合缺卯 → 卯年=填实年 → 应吉应凶取决于合局的喜忌              │
└──────────────────────────────────────────────────────────────────────────┘
```

#### 二、作用类型速查

| 作用 | 含义 | 应期信号 |
|---|---|---|
| **合**（六合/三合/天干合） | 主"到" → 牵引到位 | 事物到来/人到位/机会降临 |
| **冲**（六冲） | 主"动" → 激活/破坏 | 变动/搬迁/分离/动荡 |
| **墓**（入墓） | 主"收" → 收藏/结束 | 事情收尾/收钱/入库/病入膏肓 |
| **穿**（六穿/六害） | 主"伤" → 暗损/背叛 | 暗中受损/被人坑害/隐性伤害 |

#### 三、填实应期规则

```
填实 = 六合/三合/三会/三刑中缺少的一字被流年补上

判定：
Step 1：原局+大运已有合/会/刑的部分字 → 找缺字
Step 2：流年出现缺字 = 填实年
Step 3：填实后判吉凶：
        - 合局/会局为喜 → 填实应吉
        - 合局/会局为忌 → 填实应凶
        - 刑局填实 → 一般为凶（三刑多主灾）

特殊：墓库逢空 → 开库也无用（空亡之库未经填实前无实质内容）
```

#### 四、与 §C.4.6 的关系

§C.4.6 六大规律第④为"冲合应期互补"一行概述。本节展开完整判定流程，是定应期的**首选方法**（§C.4.6-A 已明确"最实用"）。

### C.4.8 五大凶组合速查（v7.0 新增 · GP-DY-27 ⭐⭐ · 来源高派《大运流年篇》第 8 课）

> **来源**：`高派命理理论知识库/2018弟子班_大运流年篇_55页.md` 第 8 课 一~五（"羊刃聚会…财星坏印…枭神夺食…伤官见官…比劫夺财"）。
> 本节为流年断灾提供**五大经典凶组合**速查表 —— 流年形成以下任一组合即为凶年强信号。

#### 一、五大凶组合总表

| # | 凶组合 | 触发条件 | 灾类 | 轻重 |
|---|---|---|---|---|
| **①** | **羊刃聚会** | 命局羊刃 + 岁运重见/逢冲/三合刃局/刃重重 | 病伤手术·破财·官非 | 轻=破财口舌；重=死亡 |
| **②** | **财星坏印** | 流年干支=天干印/地支财（或反之）→ 地克天=重 | 失业/迁移/破财/母病/牢灾 | 地支克天干=重；天干克地支=轻 |
| **③** | **枭神夺食** | 柱有枭逢食年 / 柱有食逢枭年 / 枭食夹日 | 灾病不顺·牢狱 | 双枭/三枭夺食=灾重；食旺枭难夺=灾轻 |
| **④** | **伤官见官** | 原有伤官逢官年 / 原有官逢伤年 | 女=夫妻不和；男=惹事生非·牢灾 | 有伤逢官=灾轻；有官逢伤=灾重 |
| **⑤** | **比劫夺财** | 财星透干无官护 + 逢岁运劫财透干 | 破财 | 有官护=不惧比劫；无官护=必破 |

#### 二、各组合细则

**① 羊刃聚会**
```
《命藏经》："羊刃聚会损六亲，不入地府恐失财"
触发形式：刃见刃（伏吟）/ 刃逢冲 / 三合刃局 / 日坐刃逢冲
日坐羊刃逢冲 → 夫妻有灾
```

**② 财星坏印**
```
结构：流年干支一为印一为财 → 互克
判轻重：地支克天干（下克上）= 重；天干克地支（上克下）= 轻
忌讳点：财为忌克印为用 = 最凶 → 失学/失业/母病/牢灾
```

**③ 枭神夺食**
```
四种触发：
1. 柱有枭 + 岁运逢食 → 有灾
2. 柱有食 + 岁运逢枭 → 有灾/不顺
3. 枭食同透/枭食夹日 → 一生不顺事多；夹日=牢狱
4. 柱中无枭无食 + 岁运逢枭食 → 灾轻（无根基）

加重：双枭/三枭夺食 = 灾重
减轻：食旺枭难夺 = 灾轻
特殊：柱有枭无食 + 逢枭年 = 本人有病（不夺食但枭旺伤身）
```

**④ 伤官见官**
```
盲派口诀："伤官伤官端药罐，伤是小鬼胎中堕，旧仇新恨找上门"
男命：惹事生非/顶撞领导/打架斗殴/违法乱纪 → 严重牢灾
女命：夫妻不和/闹纠纷/情绪化

判轻重：有伤逢官（伤是旧有的·官是新来的）= 灾轻
        有官逢伤（官是旧有的·伤来犯官）= 灾重
```

**⑤ 比劫夺财**
```
条件：财星透干 + 无官星护卫 + 逢岁运劫财透干 = 破财
解法：有官星护卫（官制比劫）→ 即使逢比劫年也不惧
```

### C.4.9 发财应期 6 条铁律（v7.0 新增 · GP-CY-08 ⭐⭐ · 来源高派《财运篇》第 4 课）


### C.4.10 破财应期 6 条铁律（v7.0 新增 · GP-CY-15 ⭐ + GP-CS-24 ⭐⭐ · 来源高派《财运篇》第 7 课 + 《财运事业断法》第 18 课）

#### 一、6 条破财应期铁律

| # | 铁律 | 核心逻辑 |
|---|---|---|
| **①** | 成格局逢岁运破格 = 破财 | 格局被破 → 事业根基动摇 → 财运必损 |
| **②** | 财星外透无官护 + 逢比劫岁运 = 破财 | 无官制比劫 → 比劫明抢 → 必破 |
| **③** | 原局官杀旺无制 + 逢财运 = 破财 | 财生杀 → 杀更旺 → 破财又招灾 |
| **④** | 专旺格逢财星运 = 破财 | 专旺忌财（逆势）→ 财来犯旺 → 破 |
| **⑤** | 岁运逢岁破/破碎/大耗 = 极易破财 | 凶煞年份财星受损（神煞层面辅助） |
| **⑥** | 财局合化成比劫 = 才来财去 | 地支合成财局（来钱）+ 天干合化比劫（漏财）→ 赚了又花光 |

#### 二、核心铁律

```
┌──────────────────────────────────────────────────────────────────────────┐
│  铁律 §C.4.10-A：破财 6 条与发财 6 条（§C.4.9）互为镜像                    │
│    发财=引动发财基因 / 破财=引动破财基因 → 同一年可能"一边发一边破"        │
│  铁律 §C.4.10-B：第①条与 §C.2.9 大运修改格局联动                          │
│    格局被破=最严重的破财 → 非简单"花钱"而是"根基动摇"                    │
│  铁律 §C.4.10-C：第②条是 §C.4.8 ⑤比劫夺财的应期落地                      │
│    有官护=不惧 → 无官护+逢比劫透干=铁断破财                               │
│  铁律 §C.4.10-D：第⑥条"才来财去" → 表面有钱进但最终留不住                │
│    地支合财局=有收入 + 天干化比劫=同步漏财 → 净收益≈0                      │
└──────────────────────────────────────────────────────────────────────────┘
```

#### 三、判定流程

```

Step 4：命中条数评估：

        - ≥ 2 条 → 大破财年（★★★★）
        - + 凶组合 → 破财兼灾（★★★★★）
```

### §C.7+.3 小限推步法（GP-MG-10 · 来源第 4 课 §小限求法）

> 原文锚点：「由命宫位上起生年支，逆数到流年年支…用数字加减则很方便」

**铁律 §C.7+.3-A**：小限 = 流年在命宫十二宫的**动态投影**。

**排法（二法取一）**：
1. **逆数法**：从命宫起生年支 → 逆数到流年年支 = 该年小限所在地支
2. **公式法**：12 + 生年地支序数 - 流年地支序数 = 小限位置（结果 > 12 则 -12）

**使用规则**：
- 小限所临之神煞 = 当年吉凶主调
- 小限与四柱某柱刑冲穿 = 该六亲有事
- 小限逢空亡 = 该年事多虚

### §C.7+.5 月限推步法（GP-MG-12 · 来源第 4 课 §如何推月限）

**排法**：

2. 五虎遁配天干 = 月限干支

### §C.7+.7 长生十二宫六亲联动实战法（GP-MG-14 · 来源第 5 课 §三.1 释意）

**操作步骤**：
1. 确定目标六亲天干（如妻星乙木/七杀丙火/正印壬水等）
2. 查该天干的长生十二宫表（阳干顺行/阴干逆行）

4. 能量状态 = 长生/帝旺 = 该六亲力量强；= 绝/死/病 = 该六亲力量弱

### §C.8+.5 寿元星确定四法（GP-CH-10 ⭐⭐）

> "确定哪个是寿元星…寿元星的具备条件"——寿限专题

**寿元星优先级**（按序取第一个满足条件者）：

| 优先级 | 寿元星 | 取用条件 |
|---|---|---|
| ① | **食神**（无食用伤官） | 日主之泄=寿命之象征（泄则流通=生命力） |
| ② | **印绶** | 护身符=母亲之庇护（生我=保命） |
| ③ | **禄神** | 日主之禄=身体本身（禄=体能/健康） |
| ④ | **财星** | 养命之源（以上皆无时用财=最后一道防线） |

**寿元星三条件**（必须同时满足）：
1. 命中**少而精**（多了分散无力）
2. **靠近日主**或兼月令（远了无力护身）
3. **通根得气**（有根才有实力）

### §C.8+.7 六亲有灾八条判断规律（GP-CH-09 ⭐）
