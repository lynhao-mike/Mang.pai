## §Q.4 取象铁律（三条）

> 来源：`盲派命理AI分析师提示词_v1.md` 第二部分 + `盲派命理AI分析师提示词_v2.0.md` 第一部分『体用论命入手七步法』第三步『寻根找出处』+ 第四步『找渠道看工具』。

### 铁律 Q.4-A · 天干必找地支根源
- 干无根 → 浮象，象意不兑现
- 干有根 → 实象，象意可兑现
- 根受伤（被冲 / 穿 / 合走 / 入墓）→ 兑现窗口被截
- 根再透 → 同象重见，加倍兑现
- 配套：合关系力量对比（v2.0 第四部分）—— 1:1 / 1:多 / 多:1 / 分组四态决定象意能否成立。

### 铁律 Q.4-B · 我宫透出 = 命主特使
- 日支（我宫）所藏 → 透到天干 = 命主亲自参与该象
- 透到时干 → 命主主导晚年事
- 透到月干 → 命主主导事业 / 单位事
- 透到年干 → 命主对外（国家 / 社会）有担当
- 透出之物 = 命主"派出去"的角色

### 铁律 Q.4-C · 年月透入 = 国家社会送来
- 年柱 / 月柱地支所藏 → 透到日 / 时干 = 外部送进来
- 年透到日 → 国家 / 远方馈赠 / 祖业
- 月透到日 → 单位 / 父母 / 同事影响
- 透出物 = 命主所"接收到"的资源（也可能是负担）
- 配合体用：透出物为用 = 资源；为忌 = 负担

> **三条铁律联用范式**：① Q.4-A 先看根源是否健康 → ② Q.4-B/C 看落到我宫还是他宫，方向是『派出去』还是『送进来』→ ③ 才能给象意打分（成立 / 部分成立 / 不成立）。

## §Q.5 取象自检 5 问

> 任何取象类断语在输出前必答全 5 题，**任一未通过 → 该象意必须降档或撤回**。
