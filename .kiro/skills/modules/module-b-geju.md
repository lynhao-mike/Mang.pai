## 【B.1 体用辨析（先做这一步，必须一句话定）】

### B.1.1 五种主宾关系

| 关系 | 触发判据 | 取象方向（社会角色） |
|---|---|---|
| **我去取财** | 日主主动克财 / 财在他宫不贴身 | 主动型谋利、销售、投资、创业 |
| **财来寻我** | 财星贴日 / 财在我宫 / 财得令 | 被动型受益、继承、入职大平台 |
| **我被财困** | 财多身弱 / 财坏印 | 为钱所累、健康受损、过劳 |
| **我护他星** | 印护身、食制杀、官护财 | 庇护型职业（医、师、保、律） |
| **他星攻我** | 七杀无制 / 伤官见官 / 群比夺财 | 受外部压制、官非、被动应对 |

### B.1.2 体用一句话模板

```
体用判定：日主 [日干][月令] 生于 [月份]，[得令/失令]，[体星] 为体，[用星] 为用
        ——属"[五种之一]"格局。
```

> 这一句话**是后续所有判断的总锚**。模糊或多义会导致 B.2/B.3 全部失真。

## 【B.2 格局判定（先看月令，再看透干）】

### B.2.1 八格速判流程

```
Step 1: 看月令本气透不透干
        透 → 直接以本气定格（如"月令巳火透丙 → 食神格"）
        不透 → 看月令藏干哪个透
                透中气 → 以中气定格
                透余气 → 以余气定格
                都不透 → "杂气格"，看用神在何处

Step 2: 看格成 / 格败
        成格：用神得力、护用神到位
        败格：用神被冲克 / 用神入墓不开 / 用神远离
        中和：用神在 + 但有伤损

Step 3: 看格高 / 格低
        高：用神得令 + 透干 + 不被破 + 配印护
        中：用神得地不得令 / 透干但有微损
        低：用神不得地 + 不透 / 用神被多重克破
```

### B.2.2 八格输出表

| 格 | 月令本气 | 用神 | 喜 | 忌 |
|---|---|---|---|---|
| 正官格 | 官星 | 官 | 印护、财生 | 伤、刑冲、合杀 |
| 七杀格 | 七杀 | 食神 / 印 | 食制、印化 | 财党杀、杀重无制 |
| 正印格 | 正印 | 官 / 印 | 官生印、印护身 | 财坏印 |
| 偏印格 | 偏印 | 食 / 杀 | 杀化印、印生身 | 食被夺 |
| 食神格 | 食神 | 食 / 财 | 食生财 | 偏印夺食 |
| 伤官格 | 伤官 | 财 / 印 | 伤生财、印护 | 见官、伤多无制 |
| 正财格 | 正财 | 财 / 食 | 食生财 | 比劫夺财 |
| 偏财格 | 偏财 | 财 / 食 | 食生财、官护 | 群比争财 |

> **从化格、专旺格、化气格** —— 出现率低，先按八格走，主动确认时再切。

## 【B.3 富贵贫贱级别（不打数字分，只给等级 + 推导链）】

### B.3.1 五等级框架

| 级别 | 标记 | 特征关键词 |
|---|---|---|
| **顶级（一品）** | ★★★★★ | 用神得令 + 透干 + 三合/六合护 + 大运至少两步顺、忌神被合化 |
| **上等** | ★★★★ | 用神得令 + 透干，护用神在但不全；大运两步顺即兑现 |
| **中等** | ★★★ | 格成但用神有损 / 喜忌交战；靠大运补救可达上等 |
| **中下** | ★★ | 格半成 / 用神不透 + 多见破损；靠运气也只是平稳 |
| **贫贱** | ★ | 格破 + 主要五行皆受伤 / 反背 + 大运全程不顺 |

> ⚠ **此处的"贫贱"不是道德评价，仅指**"该命格在不借助外力的情况下，物质 / 社会地位的天花板很低"**。
> 时代与个人努力可改变实际兑现（→ 配合模块 C 的"时代修正系数"）。

### B.3.2 富 / 贵 / 寿 / 名 四向分判

不是所有命都"全面富贵"。一个上等命可能"贵而不富"或"富而不贵"。

| 维度 | 主要看 | 上等条件 |
|---|---|---|
| **富** | 财星 + 食伤 | 财星得地 + 食伤生财 + 财不被夺 |
| **贵** | 官星 + 印 | 官星清纯（不混杀）+ 印护身 + 官星不被冲 |
| **寿** | 日主 + 印 | 日主气足 + 印有根 + 没有大冲克日主 |
| **名** | 食伤 + 官 / 印 | 食伤透干 + 不被夺 + 名声星（驿马 / 文昌）相配 |

| 贵 | ★★★ | 官星清但被穿 + 印护不到位 |
| 寿 | ★★★★ | 日主得令 + 印有根 + 无大冲克 |
| 名 | ★★ | 食伤透但被偏印夺 |

## 【B.4 意向识别（这个命"想做什么"）】

### B.4.1 主导意向四类

| 意向类 | 命局信号 | 取象画像 |
|---|---|---|
| **取财型** | 财星透干 + 食伤生财 | 商人、自雇、销售、投资 |
| **求名型** | 官印相生 + 食伤名声 | 学者、技术专家、官员、艺人 |
| **守护型** | 印重 + 食伤制杀 | 医、教、律、保、家族守业 |
| **冲撞型** | 杀刃 / 伤官旺无制 | 军警、运动员、改革派、极端创业 |

> 意向≠职业，而是"驱动模式"。同一个意向在不同时代会落到不同行业。

### B.4.2 意向与做功的对照（与模块 A 联动）

```
意向：取财型
    ↓
做功核心应是："食伤 → 财" 这条链路通畅
    ↓
若模块 A 显示"食伤被穿"或"财被合化" → 意向受阻 → 等级下调一级
```

## 【B.6 学历层次细分制（v6.0.2 新增 · 来自 #CASE-001 REVISE-003）】

> 此前模型仅区分"本科 vs 硕博"，粒度过粗。v6.0.2 起按下表细分：

### B.6.1 六级学历框架

| 层次 | 标记 | 基础条件 |
|---|---|---|
| **985 / 顶校** | ★★★★★ | 印透干得令 + 文昌不空 + 食神制伤 + 月令为印或官 |
| **211 / 重点** | ★★★★ | 印透干 + 文昌不空 + 杀印相生（任一减分项 → 降一级） |
| **一本** | ★★★ | 印透或印得令（二选一）+ 文昌相对清纯 |
| **二本** | ★★ | 印藏不透 + 文昌带瑕（空亡 / 被穿）但仍有用 |
| **三本 / 民办** | ★ | 印藏不透 + 文昌空亡或被合 + 杀轻伤重（学业靠"考"上不去靠"复读"撑） |
| **专科 / 中专** | ◯ | 文昌全失 + 印星彻底受伤 + 多次复读仍勉强 |

### B.6.2 减分项扣减规则

每出现一项 → 在基础层次上**下调一级**：

- 文昌空亡且未被填实
- 杀轻伤重（合杀失衡）
- 十恶大败 / 阴差阳错（古法减分）
- 印星不透
- 比劫夺食（食伤路被堵）
- 月令为忌神且不被合化

### B.6.3 高考兑现年份判定

> 仅"方向"判断，年份 ±1 弹性。和模块 C 协同。

- **顺利上岸年**：印透流年 + 文昌得用 + 大运无重大冲克
- **失利复读年**：印星受冲 / 文昌被穿 / 流年透出忌神（如本案 2004 甲申，甲为正官但申冲寅破印根）
- **典型组合**："失利年 + 1 = 复读上岸年"（本案 2004 → 2005）
