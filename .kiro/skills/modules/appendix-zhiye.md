## §Z.1 11 种职业画像（v1 全量迁入 + 精炼）

> 整节来源：`盲派命理AI分析师提示词_v1.md` 第六部分『11 种职业特征识别库』。
> 每条职业 = 驱动结构（格局组合）+ 取象证据（地支 / 天干字象）。**没有命中至少 2 条核心特征不得直断该职业**。

### 教师
- **驱动结构**：木火旺 = 文科 / 金水旺 = 理科；伤官配印 / 食神佩印 / 印库 + 食伤库
- **取象证据**：戌为学校象（戌 = 寺观 / 校舍）
- **典型组合**：印透 + 食伤透 + 戌字现
- 来源：v1.md 第六部分『教师』节。

### 公务员
- **驱动结构**：印透时上坐官星 / 正官格配印 / 伤官配印格 / 调候得用
- **取象证据**：我宫与年月有感应 = 吃皇粮（年月透入我宫）
- **典型组合**：官印相生且根稳；或杀印组合（武职体制）
- 来源：v1.md 第六部分『公务员』节。
- **细分见 §Z.2**（v2.0 补充：一把手 / 参谋 / 业务骨干 / 技术官员 / 纪检 / 清贵）。

### 医生
- **驱动结构**：『辛乾巽丙，术士医流』 → 辰巳戌亥 + 辛丙组合；天医星显；日主『病地』为用神
- **取象证据**：庚申 = 外科 / 丑午穿 = 手术 / 辰庚申辛 = 医院 / 辛 = 药 / 卯酉 = 针
- **典型组合**：辛 + 丙 + 戌 / 辰 / 巳 / 亥之三件套
- 来源：v1.md 第六部分『医生』节。

### 银行
- **驱动结构**：带酉丑 / 金与财星组合被我控 / 寅申冲 / 卯申合（收发之意）
- **取象证据**：酉 = 银行 / 钟表 / 现金；丑 = 金库；申 = 金融机构
- **典型组合**：酉丑双现 + 财官生
- 来源：v1.md 第六部分『银行』节。

### 房地产
- **驱动结构**：杀制比劫 / 印星发达 / 木土组合 / 食伤库 + 印库刑合
- **取象证据**：戊土 = 大坝 / 高楼；辰戌 = 库地；木 = 建筑（梁）
- **典型组合**：戊 + 辰戌 + 比劫被制
- 来源：v1.md 第六部分『房地产』节。

### 律师
- **驱动结构**：食伤旺（口才）+ 印星木（法律）+ 官杀配合 + 阳制阴 + 亥字多
- **取象证据**：辛 = 法律 / 刑具；亥 = 文墨 / 天门；卯 = 文书
- **典型组合**：亥多 + 辛透 + 食伤旺
- 来源：v1.md 第六部分『律师』节。

### 公检法
- **驱动结构**：阳刃驾杀 / 食神制杀 / 伤官佩印 / 伤官驾杀 / 刑穿害 / 阳制阴 / 庚辛申酉
- **取象证据**：庚辛申酉 = 武器 / 刑具；刑穿害 = 执法手段
- **典型组合**：庚 + 申 + 阳刃驾杀 + 多刑穿
- 来源：v1.md 第六部分『公检法』节。

### 生意人
- **驱动结构**：食伤生财 / 财逢库 / 合多 = 和气生财 / 『寅申庚甲，商路吏人』
- **取象证据**：合多 = 善协调；财库 = 存财能力；寅申冲 = 流通
- **典型组合**：食伤透 + 财星根重 + 寅 / 申 / 庚 / 甲多见
- 来源：v1.md 第六部分『生意人』节。

### 小偷
- **驱动结构**：印被坏（缺道德）+ 官被坏（无法律意识）+ 比劫旺无制 + 取财手段不正（冲穿刑暗合墓）
- **取象证据**：印冲 / 印破 + 官杀缺位 + 比劫劫财
- **典型组合**：印破 + 官损 + 比劫多 + 财被冲穿合走
- 来源：v1.md 第六部分『小偷』节。

### 体力劳动者
- **驱动结构**：比劫禄与财官联 / 官杀多无制 / 身弱不从
- **取象证据**：比劫 = 劳动力；身弱被官杀压 = 出力卖苦
- **典型组合**：官杀重 + 印不出 + 比劫禄被夺
- 来源：v1.md 第六部分『体力劳动者』节。

### 财务
- **驱动结构**：财合印 / 财冲印 / 日坐财库 / 财星伏吟 / 官杀下配财星 / 水 = 计算
- **取象证据**：财与印的紧密互动 = 账目；水 = 流水 / 计算
- **典型组合**：日支财库 + 印旺 + 水透
- 来源：v1.md 第六部分『财务』节。

## §Z.2 公务员 6 细分画像（v2.0 补充）

> 来源：`盲派命理AI分析师提示词_v2.0.md` 第七部分『v2.0 补充：公务员细分画像』+ 第三部分『仕途天花板判定法』。
> **与 §Z.1 公务员的关系**：先用 §Z.1 判定『是否吃皇粮』，再用本节细分『干哪一档 / 哪条线』。

### 一把手型
- **核心特征**：官印相生 + 日主旺 + 无忌；力量 1:1 均衡
- **典型格局**：正官格配印 + 印根重 + 官无伤
- **驱动模式**：决断 / 全局 / 担责
- 来源：v2.0 第七部分公务员细分表第 1 行。

### 二把手 / 参谋型
- **核心特征**：伤官合杀 + 伤官**微微强于**杀；典型即 **#CASE-001 双伤合一杀**结构
- **典型格局**：伤官透干 + 杀根浅 + 我宫见伤
- **驱动模式**：辅佐 / 协调 / 给方案；"大事办不了，小事可协调"
- 来源：v2.0 第七部分公务员细分表第 2 行 + 案例 001 已晋级（详见 `cases/career.md` 与 `module-b-geju.md`）。

### 业务骨干型
- **核心特征**：食神生财 + 带库
- **典型格局**：食神透 + 财根重 + 财库藏
- **驱动模式**：执行 / 创造 / 业绩；窗口岗 / 项目岗
- 来源：v2.0 第七部分公务员细分表第 3 行。

### 技术官员型
- **核心特征**：伤官佩印 + 印得地；有印透但**不及伤官**
- **典型格局**：伤官 + 印 + 食伤强 / 印中等
- **驱动模式**：技术决策 / 专业管理；总工 / 总师 / 处级技术岗
- 来源：v2.0 第七部分公务员细分表第 4 行。

### 纪检执法型
- **核心特征**：阳制阴 + 金旺无水泄；庚辛克甲乙
- **典型格局**：庚辛透干 + 阳刃驾杀 + 木被克
- **驱动模式**：监督 / 执法 / 肃杀
- 来源：v2.0 第七部分公务员细分表第 5 行。

### 清贵文职型
- **核心特征**：华盖 + 德秀 + 文昌齐现；案例 001 副特征
- **典型格局**：华盖透 + 德秀（月德 / 天德合）+ 文昌得用
- **驱动模式**：清贵 / 文字 / 秘书 / 研究室；位次不高但风评好
- 来源：v2.0 第七部分公务员细分表第 6 行 + 案例 001。

### 仕途天花板判定法（v2.0 第三部分补丁）

在判断公务员仕途级别**前**必须做合关系力量对比：

```
IF 格局为"伤官驾杀" OR "食神制杀" OR "官印相生":
  分析乙/庚/甲/丙等关键星位的力量对比
  IF 1:1 均衡            → 格局有效 → 正常判级
  IF 1:多 失衡（杀轻/官轻/印轻被多方合泄）→ 天花板下调 1-2 级
  IF 多:1 失衡（杀重/官重压身）→ 天花板下调 1 级且过程艰险
```

> 来源：`盲派命理AI分析师提示词_v2.0.md` 第三部分『仕途天花板判定法』。
> 配套：主入口 §3.0 铁断前提自检 → 任何『升至 X 级』铁断必须带 `【前提：命主当前级别为 ××】`。

## §Z.3 职业识别 5 步法

> 来源：`盲派命理AI分析师提示词_v1.md` 第三部分一节『事业专题（5 步断职业法）』+ `盲派命理AI分析师提示词_v2.0.md` 第六部分（同名）。

```
Step 1：我宫（日支）有官杀？      → 该柱为职业柱
Step 2：我宫无官杀有官杀库？      → 库柱为职业柱
Step 3：我宫全无？               → 看他宫官杀与时柱有无关系
Step 4：以上都无？               → 看财星 / 财库
Step 5：以上全无？               → 看成格星
```

**职业柱+左柱秘法**：
- **职业柱定"类型"**（公务员 / 教师 / 商人）
- **左柱（紧邻职业柱左侧）定"性质"**（哪一行 / 哪一岗）

**5 步法 + §Z.1 / §Z.2 联用范式**：
1. Step 1-5 → 锁定**职业柱**与**左柱**
2. 再用 §Z.1 11 种职业画像 → 验证**驱动结构**与**取象证据**至少各 1 条
3. 命中公务员 → 进 §Z.2 6 细分判定**仕途档位**
4. 配合 §Z.4 时代落地映射 → 给出**当下行业**的具体落地

## §Z.4 驱动模式 → 时代落地映射规则

> 同一驱动模式在不同时代会落到**不同行业**。本节把 §Z.1 的『传统画像』映射到现代场景。
> 原则：**驱动结构不变，落地行业随时代而变**；模型在断职业时**禁止只停在 1990 年代落地**。

| 驱动模式（v1 原貌） | 1990 年代落地 | 2000 年代落地 | 2010 年代落地 | 2020 年代落地 |
|---|---|---|---|---|
| 食伤生财（取财型） | 个体户 / 倒爷 | 互联网创业 / 电商 | 移动 App / 微商 | 自媒体 / 直播带货 / 内容创业 |
| 伤官配印（教育文化型） | 中小学教师 / 编辑 | 大学讲师 / 出版 | 培训机构 / MCN | 知识付费 / 在线教育 / 课程主理人 |
| 官印相生（公务员型） | 党政机关 | 公务员 + 事业编 | 选调生 / 国企管理 | 体制内 + 智库 + 政策研究 |
| 庚辛申酉（金 / 法 / 武） | 公检法 / 部队 / 银行 | 律所 / 银行业 / 安保 | 证券 / 投行 / 仲裁 | 合规官 / 反洗钱 / 网络安全 |
| 辛 + 丙 + 辰巳戌亥（医） | 医院 / 卫生院 | 三甲医院 / 民营医院 | 互联网医疗 / 医美 | AI 医疗 / 心理咨询 / 健康管理 |
| 戊 + 辰戌（地产 / 仓储） | 包工头 / 国营建筑 | 房地产开发 | 物业 / 商业地产 | 物流仓储 / 数据中心 / 仓配 |
| 食伤旺 + 亥（律师 / 文字） | 律师 / 法务 | 互联网法务 / 知产 | 数据合规 / 跨境法务 | AIGC 合规 / 数字版权 / 网络仲裁 |
| 比劫旺 + 取财不正（小偷型） | 小偷 / 黑灰产 | 黑灰产网赚 | 网络诈骗 / 套现 | 跨境黑产 / 加密货币诈骗 |
| 比劫禄 + 官杀压身（体力型） | 工厂工人 | 农民工 / 建筑工 | 外卖 / 快递 / 网约车 | 平台零工 / 灵活用工 |
| 财合 / 冲印 + 水（财务型） | 国企会计 | 上市公司财务 | 财务 BP / FA | CFO / 数字财税 / RPA 财务 |

> **铁律 §Z.4-A**：断职业时必须**至少跨两个时代**给出落地，避免输出『商人 / 个体户』这种与时代脱节的结论。
> **铁律 §Z.4-B**：当原局透出多种驱动模式（如 #CASE-001 既有伤官合杀又有食神生财）→ **以职业柱所在驱动为主线**，左柱驱动作为副线。

## §Z.5 职业识别自检 5 问

> 任何职业类断语在输出前必答全 5 题，**任一未通过 → 该职业判定必须降档或撤回**。

- [ ] **Z.5-1 步法问**：是否走完 §Z.3 的 5 步法，**职业柱**与**左柱**是否已锁定？
- [ ] **Z.5-2 画像问**：是否在 §Z.1 / §Z.2 中至少命中**驱动结构**与**取象证据**各 1 条？
- [ ] **Z.5-3 力量问**：本职业是否经过**力量对比四态**（1:1 / 1:多 / 多:1 / 分组）检验？是否已据此调整仕途天花板？（v2.0 第四部分 + §Z.2 末尾）
- [ ] **Z.5-4 时代问**：是否走完 §Z.4 时代落地映射，给出**当下时代**的具体行业（不停留在 1990 年代）？
- [ ] **Z.5-5 前提问**：涉及『当前级别 / 升迁 / 落马』的 ★★★★★ 铁断是否已带 `【前提：命主当前级别为 ××】`？（主入口 §3.0）

口诀：

```
五步先定柱，左柱辨性质。
画像两条命，力量四态查。
落地随时代，前提铁先打。
```

## §Z.7 不变量（格言区）

> *"职业柱定类型，左柱定性质。"*
> *"驱动结构不变，落地行业随时代而变。"*
> *"先 5 步法锁柱，再画像验证，再时代落地，最后铁断加前提。"*
> *"伤官合杀别一律断公务员一把手 —— 看力量对比四态。"*
