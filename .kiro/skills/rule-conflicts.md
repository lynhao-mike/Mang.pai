# 规律冲突待判池（Rule Conflict Pool）

> 新规律与既有规律矛盾时，进入此池等待更多案例仲裁。
> 禁止在此池的规律直接用于断命，只能作为"可能性提示"。

---

## 处理原则

```
冲突规律 → 标"争议中" → 等第2个案例
       ├── 新案支持新规律 → 旧规律降级
       ├── 新案支持旧规律 → 新规律移除
       └── 都有支持 → 拆分适用场景
```

## 当前争议

### 争议 #1：魁罡女命婚姻

| | 传统规律 | 新规律（#CASE-20260513-01） |
|---|---|---|
| **内容** | 魁罡女命克夫，婚姻不利 | 魁罡女命择偶标准高、不将就；配得上的人可美满 |
| **来源** | 《三命通会》等古籍 | 1个A级案例 |
| **支持案例数** | （未建库前无量化） | 1 (A=1.0) |
| **争议点** | 新规律是否只是"安慰性修正"？还是真有规律？ |
| **待验证方向** | 需要2个以上魁罡女命案例：①2个婚姻幸福的（支持新规律）②或1-2个婚姻崩溃的（支持旧规律）|
| **当前处理** | **两者并存，不作铁口断言**。遇魁罡女命时：提示"有此象，但现代社会有较大变数" |

### 争议 #2：十恶大败日

| | 传统规律 | 新规律（#CASE-20260513-01） |
|---|---|---|
| **内容** | 一生贫穷，财库有漏 | 少年无祖业，自立更生；中晚年金水运可富 |
| **来源** | 传统命理古籍 | 1个A级案例（基于大运分析的推论，实际财富结果尚未验证） |
| **支持案例数** | （未建库前无量化） | 1 (A=1.0，但此案命主当前33岁，晚年富否尚未兑现） |
| **争议点** | 新规律的"验证"是**未来预测**而非**已发生事实**——不能算真验证 |
| **待验证方向** | 需要命主33岁以上、已到金水大运、且财运情况已明确的案例 |
| **当前处理** | 新规律**降级为"假说"**，不计入晋级分数。必须有"43+岁已富"的实证案例才能晋级 |

## 仲裁触发

当某争议的任一方获得**累计加权分数≥1.8**（3个A级或等价）：
- 该方"胜出" → 另一方降级为"存疑"
- 胜出方从本池移出，进入主规律库
