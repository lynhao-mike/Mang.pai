## §M.0 矩阵符号定义

| 符号 | 含义 | 触发动作 |
|---|---|---|
| ☑ | **已吸收**（核心铁律 / 表 / 编号 已精炼到该模块；grep 可证）| — |
| ◐ | **部分吸收**（仅吸收了主干，细节 / 表 / 案例还在 pending）| 进 `pending-extraction-queue.md` 单列条目 |
| ☐ | **待入库**（尚未提炼为规律或模块章节）| 进 `pending-extraction-queue.md` P0/P1/P2 |
