## 五、M8 高派批命模式架构设计

### §M8.0 触发条件

用户在主入口输入以下关键词时触发 M8：
- "高派批命" / "盲派批命" / "体用论命" / "三大支柱"
- 或用户明确要求使用高派体系

### §M8.1 三大支柱联动框架

```
┌─────────────────────────────────────────┐
│            M8 高派批命模式               │
├─────────┬─────────────┬─────────────────┤
│ 支柱一  │  支柱二     │  支柱三         │
│ 体用论命│  象法取象   │  神煞定性       │
│ (GP-LF) │  (GP-XF)   │  (GP-SS/BD)     │
├─────────┼─────────────┼─────────────────┤
│ §B 格局 │  §Q 取象   │  §A 神煞        │
│ 体用得失│  干支宫位   │  吉凶定性       │
│ 我宫他宫│  太极转换   │  分级应用       │
└─────────┴─────────────┴─────────────────┘
         ↓ 三者交汇 ↓
    §C 大运流年（应期兑现层）
         ↓
    §D/E/职/健（专题分流）
```

### §M8.2 执行流程（9 步）

```
Step 1  排盘 + 真太阳时（§A 标准流程）
Step 2  体用划分：我宫（日时）vs 他宫（年月）
Step 3  格局定性：月令取用 + 清浊判 + 体用得失（§B 格局 + GP-LF 体系）
Step 4  象法取象：多点成像 + 太极点转换 + 干支带象（§Q 附录 + GP-XF 体系）
Step 5  神煞定性：必查神煞 + 分级应用 + 宫位归属（§A 神煞 + GP-SS/BD 体系）
Step 6  三大支柱交汇研判：格局+取象+神煞 → 综合画像
Step 7  大运流年推演（§C + GP-DY 体系）
Step 8  专题分流（婚姻/财运/职业/健康/学业 → §D/B/职/健/学）
Step 9  输出报告 + 自检（§3.0 反幻觉前置自检）
```

### §M8.3 与现有模式的关系

| 现有模式 | M8 高派模式 | 区别 |
|---|---|---|
| 标准批命（§3.1 默认路由）| M8 高派批命 | 标准模式以格局为主线；M8 以体用+象法+神煞三支柱并行 |
| 合婚模式（§D）| M8 可嵌套调用 §D | M8 完成 Step 6 后分流到 §D |
| 择日模式（§E）| M8 不直接触发 §E | §E 由独立入口触发 |
