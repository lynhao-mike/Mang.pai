## 一、GP-DY-06 引发忌神5条细则（§C.2.7）

### 适用性分析

丙火日主生酉月（金旺火退），身弱格局。忌神 = 金水（财官太旺耗泄日主）。

**丁亥大运（12-22岁·含高考关键期）检验**：

| 条目 | 是否触发 | 分析 |
|---|---|---|
| ① 忌神透干无根，行旺根之运 | 否 | 庚金忌神有根（酉、申），非无根 |
| ② 忌神强旺不透，行透干之运 | 否 | 庚已透年干 |
| ③ 忌神已受制服，行解救忌神之运 | **部分触发** | 庚被月干乙合制（合绊）；丁亥运丁火克庚 → 非"解救"而是"加强制约"！ |
| ④ 忌神强旺无制，行忌神大运 | 否 | 忌神已受乙合制 |
| ⑤ 用神/关键字在大运受伤 | **反向验证** | 丁亥运：亥水到位（七杀忌神新增），但同时亥卯未拱印 = 用神同步加强 → 忌喜同步增长，最终用神胜出 |

### 验证结论

**✅ 通过（反向验证）**：条目⑤的逻辑是"用神受伤=凶"。本案反向证明：**用神未受伤反而在同一大运增强（拱印局成）时，即使忌神也到位（亥水七杀），结果仍为吉**。这验证了条目⑤判断的核心逻辑——关键不在忌神是否出现，而在用神是否同步被破坏。

### 规则修正建议

> 条目⑤可补充正面表述："用神/关键字在大运**同步得到加强**，即使忌神也到位，可化凶为吉（前提：用神增量 ≥ 忌神增量）"

## 三、GP-DY-13 大运合冲4效应（§C.2.8）

### 适用性分析

**丁亥大运（12-22岁）与命局的合关系**：

```
亥（大运支）与 申（日支）= 亥申六合（暗合·水局方向）

申 在原局的十神角色：
  - 申中庚 = 偏财（忌神·耗身）
  - 申中壬 = 七杀（忌神·克身）
  - 申中戊 = 食神（喜用·制杀）

判定：申整体以偏财/七杀为主气 → 申 = 忌神宫
```

**对照4效应**：

| 效应 | 条件 | 是否匹配 |
|---|---|---|
| ① 忌神受运合绊 | 原局忌神被大运合住不动 | **✅ 匹配** — 亥合申 = 忌神宫（财杀）被合绊 → 吉 |
| ② 喜用受运合绊 | — | 否（申非喜用主宫） |
| ③ 忌神被合化为喜 | — | 亥申合非化合条件 |
| ④ 喜用被合化为忌 | — | 否 |

### 验证结论

**✅ 通过**：丁亥运亥合住日支申（忌神偏财/七杀宫），对应条目①"忌神受运合绊=吉运"。实际验证：丁亥运（12-22岁）正是命主学业黄金期（高考701），命局忌神被大运锁住，印星同步加强，学业大顺。

## 五、GP-XF-14 入墓5条件+开库3法+关库3法（§A.2.10）

### 适用性分析

**年支辰 = 水库（壬癸水之墓）**

检验"七杀壬水是否入辰墓"（5条件逐一）：

| # | 条件 | 案例003 检验 | 结果 |
|---|---|---|---|
| ① | 五行衰弱 | 壬水（七杀）在命局无天干透出，仅申中一藏 → 弱 | ✅ |
| ② | 墓库当令或得势 | 辰在年支，得月支酉金生助（金生水入库）→ 库有收纳力 | ✅ |
| ③ | 入墓五行与库有感应路径 | 壬水藏申 → 申辰半合水局 → 有路径"走向"辰库 | ✅ |
| ④ | 无冲库之力 | 原局无戌字冲辰（戌不在四柱） → 库未被冲开 | ✅ |
| ⑤ | 无透干牵引 | 壬水未透天干 → 无被合走之忧 | ✅ |

**结论：七杀壬水入辰墓 = 成立 ★★★★★**

### 开库验证（2018 戊戌年）

```
流年戊戌：
  戌冲辰 = 开库法② 冲开！
  → 辰库中壬水七杀被冲出
  → 但同年戊土食神透天干 = 食神制杀
  → 七杀出库即被制 = 大吉（学业格局成立）

实际应验：2018 高考 701 分 = 食神制杀格兑现年
```

### 验证结论

**✅ 铁证通过**：
- 入墓5条件：全部满足（壬水七杀确实入辰墓）
- 开库法②冲开：2018戊戌年戌冲辰 → 七杀出库
- 出库后的命理效应：食神同年透干制杀 → 学业大成

**规则评分建议**：本条从候选 0.6/2.4 建议升级至 1.2/2.4 🟡→🟢
