## 三、单盘命格（精确修正版）

### 3.1 格局判定

```
月令：卯木食神当令（本气乙未透）
格局：食神格（月令）+ 正印高透时柱配印 = 食神配印格（中上等）

⚠ 修正记录：初始误判为"正印格"（因庚透时干坐申禄）
   纠正：取格首看月令，月令本气食神当令 → 食神格为主格
         庚正印虽强但不在月令 → 仅为辅格
```

### 3.2 命格主线

| 层 | 结构 | 含义 |
|---|---|---|
| 核心轴 | 食神生偏财（卯乙→丁火） | 凭专业才华生外财 |
| 辅助轴 | 日干癸暗合时支申中戊（**暗合·不化**） | 对官身有引力·需流年戊出才兑现 |
| 保护层 | 庚正印坐申禄 | 体制资历/组织资源后盾 |
| 压制层 | 庚印克卯食神（印克食） | 才华被组织/规矩压制 |

> ⚠ **暗合不化铁律**：戊癸暗合存在但**不化**（明合都未必化何况暗合）。
> 平时命格主线仍是食神生偏财。仅在流年戊明出（如2028戊申）时短暂明合兑现。
> **新规律 R-004-REVISE-08**

### 4.5 正官能量投射洞察（本案独有深层发现）

```
命主命格中"戊正官"的投射路径：

  原局戊正官 = 仅在时支申中余气（最弱可能根基）
        ↓
  丈夫（现实配偶）= 待业·主持家务 = 完全无法承接"戊正官"能量
        ↓
  男方（业务搭档）= 戊戌日主·省属副总 = 戊正官的极致兑现
        ↓
  命主一生追求的"管理权力/社会地位/官身"
  → 在丈夫身上看不到兑现
  → 在男方身上看到自己想要的一切
        ↓
  这是五重磁场的根本动力之一（不只是桃花吸引·是正官能量投射）
```

> 这是最需要边界管理的命理底层：命主对男方的"吸引力"=正官能量投射+桃花磁场+日干合+同序共振四重叠加。
