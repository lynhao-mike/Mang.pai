**关键反思**：盲断中"心血管/糖尿/腰肾"三选一基本覆盖中国 50+ 岁父母的常见病，类似算命师套话，模糊兜底获得的命中率**不构成 skill 真实预测能力**。

真正的盲派功夫应该输出：
- 具体器官（颈喉甲状腺 / 心 / 肾 / 腰腿筋骨）

## 三、命理回归：父亲甲状腺癌的 4 重铁证（v6.1.2 §B.10 模板首例）

### 3.1 父亲星定位（铁证 ①）

| 项 | 内容 |
|---|---|
| 命主性别 | 乾造（丙日） |
| 父星定位 | **偏财庚**（丙日见庚为偏财）|
| 父星位置 | **年干透出**（最高位） |
| 父位（出生时父亲所在宫位）| 年柱（庚辰） |
| 高位/低位判定 | **高位** → 病在颈喉头面 |

> 铁律：年干透父星 = 父亲身病多在头面颈喉位（高位）。

### 3.2 器官锁定（铁证 ②）

```
五行 = 庚金
  ↓ 五行→器官对应表（v6.1.2 §B.10.3）
  → 肺 / 大肠 / 鼻 / 皮毛 / 骨 / 牙 / 咽喉 / 甲状腺
  
高位过滤（年干透）
  → 颈喉位优先 → 甲状腺 / 咽喉 / 食道
  
十神色彩 = 偏财
  → 偏财受克 = 肉体（财=肉）+ 父亲（父星）+ 财星（金钱）三重耗损
  
宫位辅助 = 年柱辰库
  → 辰中藏戊乙癸：戊土埋金（肿物）+ 乙木盗气（异变）+ 癸水阴湿（恶性）
  
三重交叉
  → 颈部金属性器官 + 库内异变 + 财星受克
  → 【甲状腺癌】★★★★★ 锁定
```

### 3.3 时间锁定（铁证 ③）

```
触发模式 = ① 合走（流年天干合年干父星）

2014 甲午：甲合年干庚（弱合 · 父星初动） → 暗藏前兆
2015 乙未：⭐ 乙合年干庚（强合 · 父星合走）
   + 流年地支未刑戌（潜在库震）
   + 流年地支未为木库（乙得根 → 合力增强）
   → 父星完全被合走 = 父亲发病 ★★★★★
2016 丙申：申=日支伏吟 + 父星弱（合后无救）
   → 治疗 / 手术期

大运配合：12-22 岁丁亥运
   - 丁火克庚（父星受克）
   - 亥水生乙木（合力得助）
   - 亥申合（金水交战）
→ 大运层面整体克父星
→ 与流年合力 = 时间锁定 2015 乙未
```

### 3.4 轻重判定（铁证 ④）

| 项 | 评估 | 影响 |
|---|---|---|
| 库内救应 | 辰中戊土埋金（土生金有救应）| 病可治、不致命 |
| 命主自身护卫 | 双印夹日（命主有护，但父位无护）| 仅护命主自身，不及父亲 |
| 多重克制 | 庚被乙合 + 庚坐辰被乙盗 + 大运丁火克庚 = 三重克 | 病势严重（→ 癌肿） |
| 现代医学映射 | 颈喉 + 金属器官 + 内部异变 + 三重克 | **甲状腺癌** |
| 预后判定 | 辰为水库（湿土埋金）+ 戊土救应 | **手术后稳定，10 年无复发** |

## 四、母亲腰腿痛的 4 重铁证

### 4.1 母亲星定位（铁证 ①）

| 项 | 内容 |
|---|---|
| 母星定位 | **正印乙**（丙日见乙为正印） |

| 母位 | 月柱乙酉 |
| 高位/低位 | 月柱根基位（贴日） → 慢性贴身 |

### 4.2 器官锁定（铁证 ②）

```
五行 = 乙木
  → 肝 / 胆 / 筋 / 神经 / 头发
  
位置 = 月柱坐酉死地（绝）
  → 母星根基弱
  → 木弱受金克长期化 = 慢性化
  
五行→器官低位扩展
  → 腰腿筋骨（坐死绝地受金克）
  
十神色彩 = 正印
  → 印=母 + 印=安身立命 → 母亲身体 + 长期慢性
  
三重交叉
  → 慢性筋骨腰腿痛（非急性外伤型）★★★★
```

### 4.3 时间锁定（铁证 ③）

```
触发模式 = ④ 受刑/受穿（长期化）+ ① 合走（局部加重）

长期慢性化：酉中辛金长期克月干乙（每年逢申酉月加重）
高峰加重年：
  2025 乙巳：乙得根但巳火克金（双方拉锯 → 母亲腰腿可能阶段性好转/加重）
  未来：辛酉年 / 甲申年（金克木流年）

→ 时间锁定：腰腿痛属"持续多年的慢性病"，而非"某年突发"
```

### 4.4 轻重判定（铁证 ④）

| 项 | 评估 |
|---|---|
| 长期慢性 vs 急性 | 长期慢性（坐死绝地受克 = 长期化）|
| 是否致残 | 不致残（双印透干仍有根，仅伤其末梢）|
| 现代医学映射 | 慢性腰肌劳损 / 退行性关节炎 / 风湿（非肿瘤、非中风）|
| 预后 | 不影响日常生活，但终生伴随 |

## 七、教训与反思

### 教训 #1：盲测中"半准蒙对"比"未中"更危险

> 看似命中实际是模糊兜底覆盖大概率事件 → 假命中率 → 虚高 skill 信心。

**修复**：v6.1.2 兜底词黑名单 + 三重精度强制约束，避免再用"心血管/糖尿/腰肾"模糊套话。

### 教训 #2：模型缺乏"亲属高位/低位"取象

> 父亲甲状腺癌的命理铁证其实非常清晰（父星透年干 = 颈喉位），但模型不会自动推到这一层。

**修复**：v6.1.2 §B.10.2 表格化亲属星定位 + 高位/低位扩展。

### 教训 #3：应期推演只关注命主自身，忽略亲属

> 模型主流应期推演（模块 C）以"命主自身"为视角，对"父星合走年/母星受刑年"等亲属类应期缺乏专门触发。

**修复**：v6.1.2 §B.10.4 应期推演 4 模式（合走 / 冲克 / 入墓 / 受刑穿）专门用于亲属事件。
