## 四、入库流程预告

每次新增 md 文件 → Skill 启动自检（§0）会自动告警 `⚠ 检测到 X 份新文档未入库`：

```
Step 1 识别 → 您新加的 md 出现在告警列表
Step 2 分类 → 按"主题分类"标签归入 A/B/C/D/E 模块候选
Step 3 提取 → 我帮您从原文提取候选规律并附原文锚点
Step 4 入候选库 → 写入 META/pending-extraction-queue.md
Step 5 晋级 → 案例验证 ≥2.4 分后正式回写到模块文件
```

> 协议详情见 `.kiro/skills/META/ingestion-protocol.md`。
> 元规律 META-RULE-01（来源原文佐证）+ META-RULE-02（双向同步）+ META-RULE-03（覆盖度自检）将自动生效。
