### 2.1.1 双重丁壬合化木 — 命定型缘分

```

机理：
  ├─ 丁壬合 = 经典"夫妻合"
  ├─ 化神为木 = 男方的喜用印星
  ├─ 双重合 = 双倍引力
  └─ 化神向男方有利 = 男方对此关系投入感更强

铁断：
  这是命理上最强的"夫妻合化"信号——
  双方在精神层面有"非此人不可"的引力
```

### 2.1.2 三重戊癸暗合 — 持续吸附

```
男方时干癸  ←──暗合──→  女方年支戌中戊
男方时干癸  ←──暗合──→  女方日支申中戊
男方时干癸  ←──暗合──→  女方时支辰中戊

机理：
  ├─ 戊癸合 = 真挚之合
  ├─ 三处暗合 = 三重锚点
  ├─ 全部在女方藏支 = 女方深处对男方有吸引力
  └─ 男方主动的"癸"配上女方藏起的"戊" = 慢热但深层

铁断：
  在丁壬合的"显性引力"之外，还有戊癸合的"隐性吸附"——
  即便表面冷却，深层仍会被对方吸住
  这是"分不开"型的暗能量纽带
```
