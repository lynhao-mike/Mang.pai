# 【核心理念】

> **盲派不造命，盲派优命。**
>
> 剖腹产择日 = 在医学安全窗口内，为孩子选择与父母、家族、时代气场最和谐的先天坐标。
> 不是"造富贵"，而是"避凶扶正"。
>
> 年柱由预产期年份锁死；月柱由节气锁死（绝大多数窗口不跨节）；
> 真正可调整的只有 **日柱（2-10天）** 和 **时柱（12时辰/日）**。
>
> 高德臣体系的核心不是"凑格局"——而是 **"让孩子的我宫（日时）能从年月（家国时代）顺畅获取财官"**。
> 这与传统"旺衰补缺论"有本质区别。

# 【第一部分：节气 · 子时 · 真太阳时——硬规则】

## 一、月柱以节气划分（非农历月）

**铁律：月柱变更以节气精确到分钟的时刻为准。**

| 节气 | 公历约（日） | 进入月柱 |
|---|---|---|
| 立春 | 2月3-5日 | 寅月起 |
| 惊蛰 | 3月5-7日 | 卯月起 |
| 清明 | 4月4-6日 | 辰月起 |
| 立夏 | 5月5-7日 | 巳月起 |
| 芒种 | 6月5-7日 | 午月起 |
| 小暑 | 7月6-8日 | 未月起 |
| 立秋 | 8月7-9日 | 申月起 |
| 白露 | 9月7-9日 | 酉月起 |
| 寒露 | 10月8-9日 | 戌月起 |
| 立冬 | 11月7-8日 | 亥月起 |
| 大雪 | 12月6-8日 | 子月起 |
| 小寒 | 1月5-7日 | 丑月起 |

**处理规则：**
- 若医学窗口横跨节气交接日 → 必须将窗口拆为两段，分别论月柱
- 节气交接的精确时刻必须查万年历到分钟级，不可估算
- 节气交接当天若在窗口内 → 该日需标注"上午属X月/下午属Y月"（取决于交接时刻）

## 二、子时划分——采用"夜子时"规则

**本体系采用盲派主流规则：**

| 时段 | 归属 | 日柱 |
|---|---|---|
| 23:00 - 00:00（夜子时） | 属 **当日** | 用 **当日** 日柱 |
| 00:00 - 01:00（早子时） | 属 **次日** | 用 **次日** 日柱 |

**硬规则：**
- 夜子时（23:00-24:00）：时柱天干按当日日干起子时
- 早子时（00:00-01:00）：时柱天干按次日日干起子时
- 当候选日的"子时"被选中 → 报告必须明确注明"夜子时(23:00-24:00)"或"早子时(00:00-01:00)"
- **剖腹产实操：医院通常不在 23:00-01:00 安排择期手术，子时候选一般自动排除**

## 三、真太阳时校正

**铁律：所有时辰判定必须使用真太阳时，不使用北京标准时间。**

### 校正公式

```
真太阳时 = 北京时间 + (出生地经度 - 120°) × 4分钟/度 + 时差修正值
```

- 若校正后时间距时辰边界 ≤ 10分钟 → **两个时辰都要评估**，并在报告中标注"临界时辰，请与医生精确到分钟协调"

# 【第四部分：择日总纲】

## 一、变量与常量

```
┌─────────────────────────────────────────────────┐
│ 硬常量：年柱 ← 预产期年份                       │
│ 半常量：月柱 ← 节气决定（极少数窗口跨节）       │

│ 强可调变量：时柱 ← 12时辰/日                    │
└─────────────────────────────────────────────────┘
```

## 二、高德臣体系择日优先级（硬排序，不可颠倒）

```
Priority 1：父母子女配合度 → 家不和则一切无用
Priority 2：时柱质量 → 管晚年/归宿/子女/门户
Priority 3：通路顺畅 → 财官能否从月令流到日时
Priority 4：格局清纯 → 月令取格是否成立
Priority 5：三垣和谐 → 精修层
```

## 三、盲派择日七层递进法

```
第一层  硬性禁忌筛查         → 一票否决（查表）

第三层  时柱与我宫构建        → 逐时辰评估
第四层  父母子女配合          → 对照第三部分坐标清单打分
第五层  兄姐配合检查          → 有兄姐时启用（查表）
第六层  三垣评估              → 精修
```
